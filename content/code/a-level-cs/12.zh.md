## 什么是范式?

- **编程范式**(programming paradigm)是组织程序的一种风格。
- 同一个问题可以用不同的风格来解决。
- Python 允许你在一个程序里混用多种范式。

---

## 命令式与过程式

- **命令式**(imperative)代码是一串改变程序状态的步骤。
- 它一步一步地说明**如何**做某件事(循环、赋值)。
- **过程式**(procedural)风格把这些步骤组织成可复用的**过程**和函数。

---

```python
def total(nums):
    result = 0
    for n in nums:
        result = result + n
    return result

print(total([2, 4, 6]))
```

---

## 面向对象

- **面向对象**(object-oriented)代码把数据和操作它的方法打包进对象里。
- 你把问题建模成一组相互协作的类。
- 它适合大型程序,因为每个类都是一个自成一体的部分。

---

```python
class Counter:
    def __init__(self):
        self.count = 0
    def add(self):
        self.count = self.count + 1

c = Counter()
c.add()
c.add()
print(c.count)
```

---

## 声明式

- **声明式**(declarative)代码说明你**想要什么**,而不是如何计算它。
- 由语言或库替你算出具体步骤。
- 例子:数据库查询(SQL),或 Python 的列表推导式。

---

```python
nums = [1, 2, 3, 4, 5]
print(sum(nums))                 # say WHAT: the total
print([n * n for n in nums])     # a list comprehension
```

---

## 低级语言

- **低级**(low-level)代码贴近机器:汇编语言和机器码。
- CPU 直接运行它,所以速度快,但很难编写。
- 像 Python 这样的高级语言,在运行前会被翻译成它。
