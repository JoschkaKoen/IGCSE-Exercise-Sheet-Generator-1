## 图

- **图**(graph)是一组由**边**(edge,连接)连起来的**节点**(node,也叫顶点 vertex)。
- 图用来给网络建模:朋友关系、城市间的道路、网页之间的链接。
- 树其实是一种特殊的图;一般的图可以有环,每个节点也可以有很多条连接。

---

## 表示一个图:邻接表

- 一种常见的方式是**邻接表**(adjacency list):一个字典,把每个节点映射到它的**邻居列表**。
- `graph["A"]` 就是与 `A` 直接相连的节点列表。
- 当每个节点只有少量连接时,这种方式很紧凑。

---

```python
# A graph as a dict: each node maps to its list of neighbours
graph = {
    "A": ["B", "C"],
    "B": ["A"],
    "C": ["A"],
}
print(graph["A"])    # ['B', 'C']  -- A connects to B and C
print(graph["B"])    # ['A']
```

---

## 添加一条边

- 在**无向**(undirected)图里,一条边 `A—B` 是双向的。
- 所以添加它意味着:把 `B` 追加到 `A` 的列表里,**并且**把 `A` 追加到 `B` 的列表里。
- 如果某个节点是新的,先用一个空列表把它建起来。

---

```python
def add_edge(graph, a, b):
    if a not in graph:
        graph[a] = []
    if b not in graph:
        graph[b] = []
    graph[a].append(b)
    graph[b].append(a)

graph = {}
add_edge(graph, "A", "B")
add_edge(graph, "A", "C")
print(graph)    # {'A': ['B', 'C'], 'B': ['A'], 'C': ['A']}
```

## 遍历邻居

- 要从一个节点出发探索,你就读出它的邻居列表,逐个访问。
- 不在图里的节点**没有**邻居 —— 返回一个空列表,而不是报错。
- 这个邻居查找是更大任务(比如搜索整张图)的第一步。

---

## 有向 vs 无向

- 在**无向**图里,一条边是双向的(比如朋友关系)。
- 在**有向**(directed)图里,一条边只指向一个方向(单行道,或"关注"链接)。
- 对有向边,你只在**一个**方向上追加。
