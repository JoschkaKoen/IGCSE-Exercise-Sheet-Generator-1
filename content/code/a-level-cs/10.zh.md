## 类是一张蓝图

- **类**(class)定义一种新的事物 —— 一个用户自定义类型。
- **对象**(object)是用这个类造出来的一个实例。
- 一个 `Dog` 类可以造出许多狗对象,每个都有自己的数据。

---

## __init__ 与属性

- `__init__` 是一个特殊方法,在你创建对象时运行。
- `self` 指代正在被创建的那个对象。
- 存放在 `self` 上的值就是对象的**属性**(attribute,它的数据)。

---

```python
class Point:
    def __init__(self, x, y):
        self.x = x
        self.y = y

p = Point(2, 5)
print(p.x)
print(p.y)
```

---

## 方法

- **方法**(method)是定义在类内部的函数。
- 它的第一个参数永远是 `self`,即它所操作的对象。
- 用点号来调用它:`object.method()`。

---

```python
class Greeter:
    def __init__(self, name):
        self.name = name

    def hello(self):
        return "Hi, " + self.name

g = Greeter("Sam")
print(g.hello())
```

## 枚举类型

- **枚举类型**(enumerated type)是一组固定的、有名字的值。
- `from enum import Enum` 让你把它定义成一个类。
- 这比使用零散的数字(如 `1`、`2`、`3`)更清晰。

---

```python
from enum import Enum

class Color(Enum):
    RED = 1
    GREEN = 2
    BLUE = 3

print(Color.GREEN)
print(Color.GREEN.value)
```

## 用剑桥伪代码表示

- `CLASS` 代码块列出它的数据和一个 `NEW` 构造函数。

```text
CLASS Point
    PRIVATE X : INTEGER
    PRIVATE Y : INTEGER

    PUBLIC PROCEDURE NEW(startX : INTEGER, startY : INTEGER)
        X ← startX
        Y ← startY
    ENDPROCEDURE
ENDCLASS

DECLARE p : Point
p ← NEW Point(2, 5)
```
