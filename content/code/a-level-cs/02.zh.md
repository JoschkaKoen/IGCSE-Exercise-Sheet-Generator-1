## 为什么需要文件?

- 程序结束时变量就消失了。**文件**(file)把数据保存在磁盘上。
- 文件让你保存结果,以后再读回来。
- 文件也能存放比你手动输入多得多的数据。

---

## 打开、使用、关闭

- `open(name, mode)` 打开一个文件。**模式**(mode)说明你要做什么。
- `"r"` 读取,`"w"` 写入一个全新文件(替换旧内容),`"a"` 追加到末尾。
- 每个打开的文件,用完都必须**关闭**。

---

## `with` 代码块

- `with open(...) as f:` 会打开文件,并**替你关闭它**。
- 所有操作都写在缩进的代码块里。
- 这是最安全的方式 —— 文件绝不会被忘记关闭。

---

```python
with open("greeting.txt", "w") as f:
    f.write("Hi there\n")

with open("greeting.txt", "r") as f:
    print(f.read())
```

---

## 逐行读取

- `f.read()` 把**整个**文件作为一个字符串返回。
- `for line in f:` 会**一行一行**地读取。
- 每行都带着换行符,所以用 `line.strip()` 把它去掉。

---

```python
with open("fruit.txt", "w") as f:
    f.write("apple\npear\n")

with open("fruit.txt", "r") as f:
    for line in f:
        print(line.strip())
```

## 用剑桥伪代码表示

- 考试使用 `OPENFILE`、`WRITEFILE`、`READFILE` 和 `CLOSEFILE`。

```text
OPENFILE "notes.txt" FOR WRITE
WRITEFILE "notes.txt", "Hello"
CLOSEFILE "notes.txt"

OPENFILE "notes.txt" FOR READ
READFILE "notes.txt", line
CLOSEFILE "notes.txt"
OUTPUT line
```
