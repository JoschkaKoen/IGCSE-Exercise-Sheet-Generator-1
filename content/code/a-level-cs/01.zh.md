## 循环里套循环

- 你可以把一个循环放进另一个循环**里面**。
- 外层循环每跑一轮,内层循环就**完整**地跑一遍。
- 所以外层 2 次、内层 3 次,循环体会运行 2 × 3 = 6 次。

---

```python
for row in range(2):
    for col in range(3):
        print(row, col)
```

---

## 列表里的列表

- **二维列表**(2-D list)是一个其元素本身也是列表的列表。
- 它用来表示网格或表格:由若干行的若干列组成。
- 用两个索引访问某个单元格:`grid[row][col]`。

---

```python
grid = [[1, 2], [3, 4]]
print(grid[0][1])
print(grid[1][0])
```

---

## 遍历网格

- 用嵌套循环访问每个单元格。
- 外层循环走**行**;内层循环走**列**。
- `for row in grid:` 每次取出一行(一个列表)。

---

```python
grid = [[1, 2], [3, 4]]
for row in grid:
    for value in row:
        print(value)
```

---

## 用剑桥伪代码表示

- 二维列表就是考试里的**二维数组**(2-D array),写作 `ARRAY[1:rows, 1:cols]`。

```text
DECLARE grid : ARRAY[1:2, 1:3] OF INTEGER
FOR r ← 1 TO 2
    FOR c ← 1 TO 3
        OUTPUT grid[r, c]
    NEXT c
NEXT r
```
