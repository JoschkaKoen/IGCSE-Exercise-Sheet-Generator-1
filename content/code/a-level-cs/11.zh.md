## 在已有的类上扩展

- **继承**(inheritance)让一个新类复用一个已有的类。
- 新类是**子类**(subclass);原来的类是**父类**(superclass)。
- 写 `class Bird(Animal):` —— `Bird` 会获得 `Animal` 的所有方法。(考试伪代码里写作 `CLASS Bird INHERITS Animal`。)

---

## 用 super() 调用父类

- 子类可以在 `__init__` 里添加自己的属性。
- 用 `super().__init__(...)` 先运行父类的 `__init__`。
- 这样就不必重复父类的初始化代码。

---

```python
class Animal:
    def __init__(self, name):
        self.name = name
    def describe(self):
        return self.name + " is an animal"

class Bird(Animal):
    def fly(self):
        return self.name + " flies"

b = Bird("Robin")
print(b.describe())   # inherited from Animal
print(b.fly())        # new in Bird
```

## 重写方法

- 子类可以**重写**(override)方法:定义一个同名的方法。
- 这时会使用子类的版本,而不是父类的。
- 这让每个类都能有自己的行为方式。

---

```python
class Animal:
    def speak(self):
        return "..."

class Cow(Animal):
    def speak(self):
        return "Moo"

print(Cow().speak())
```

## 封装

- **封装**(encapsulation)就是把对象的数据安全地保存在它内部。
- Python 没有真正的"私有",但名字前加下划线(`self._balance`)表示"外部不要直接动它"。
- 其他代码应通过**方法**来读取或修改数据,而不是直接伸手进去。

---

## 多态

- **多态**(polymorphism)指同一个方法名,在不同类里有不同的行为。
- 一个循环可以对许多对象调用 `.speak()`,而无需知道它们的确切类型。
- 每个对象都会以自己的方式做出响应。

---

```python
class Duck:
    def speak(self):
        return "Quack"

class Cow:
    def speak(self):
        return "Moo"

for animal in [Duck(), Cow()]:
    print(animal.speak())
```
