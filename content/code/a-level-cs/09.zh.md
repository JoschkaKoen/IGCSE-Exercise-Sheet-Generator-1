## 调用自己的函数

- **递归**(recursion)就是一个函数调用**它自己**。
- 每次调用都应处理这个问题的一个**更小**的版本。
- 做对了的话,问题会不断缩小,直到简单到可以直接解决。

---

## 基本情形与递归情形

- **基本情形**(base case)是最简单的情形 —— 它让递归停下来。
- **递归情形**(recursive case)在更小的输入上再次调用这个函数。
- 没有基本情形,函数就永远停不下来(会出错)。

---

```python
def countdown(n):
    if n == 0:
        print("Go!")
        return
    print(n)
    countdown(n - 1)

countdown(3)
```

---

## 一个例子:阶乘

- 阶乘 `n!` 表示 `n × (n-1) × ... × 1`。
- 用递归形式表示:`n! = n × (n-1)!`,基本情形是 `0! = 1`。
- 每次调用都把 `n` 乘上小一号的阶乘。

---

```python
def factorial(n):
    if n == 0:
        return 1
    return n * factorial(n - 1)

print(factorial(4))
```

---

## 计算机是怎么运行它的

- 每次调用在等待内层调用时,都被暂存在**调用栈**(call stack)上。
- 当基本情形返回后,被暂停的调用会一个接一个地完成。
- 调用太多会让栈溢出 —— Python 会抛出 `RecursionError`。

---

## 用剑桥伪代码表示

- 递归函数会清楚地写明它的基本情形和递归情形。

```text
FUNCTION Factorial(n : INTEGER) RETURNS INTEGER
    IF n = 0 THEN
        RETURN 1                       // base case
    ELSE
        RETURN n * Factorial(n - 1)    // recursive case
    ENDIF
ENDFUNCTION
```
