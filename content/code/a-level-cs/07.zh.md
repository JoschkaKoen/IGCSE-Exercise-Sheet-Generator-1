## 在列表中查找

- **查找**(searching)就是判断某个值是否在列表中,以及在哪里。
- 通常的答案是这个值的**索引**,如果不存在则返回 `-1`。
- 两种经典方法是线性查找和二分查找。

---

## 线性查找

- 从头开始,依次检查每个元素。
- 一旦找到匹配项就停下。
- 它对**任何**列表都有效,无论是否已排序。

---

```python
names = ["Sam", "Mia", "Leo"]
target = "Mia"
found = -1
for i in range(len(names)):
    if names[i] == target:
        found = i
print(found)
```

---

## 二分查找

- 二分查找需要一个**已排序**的列表。
- 看**中间**那个元素。如果它就是目标,就停下。
- 如果目标更小,就在左半边找;更大就在右半边找。

---

```python
data = [2, 4, 6, 8, 10]
target = 8
low = 0
high = len(data) - 1
found = -1
while low <= high:
    mid = (low + high) // 2
    if data[mid] == target:
        found = mid
        break
    elif data[mid] < target:
        low = mid + 1
    else:
        high = mid - 1
print(found)
```

---

## 比较两者

- 线性查找可能要检查**每一个**元素 —— 对长列表来说很慢。
- 二分查找每一步都丢掉**一半**列表,所以快得多。
- 但二分查找只有在数据已经**排好序**时才有效。

---

## 用剑桥伪代码表示

- 注意 `DIV` 是整数除法(对应 Python 的 `//`)。

```text
// Linear search
FOR i ← 0 TO LENGTH(list) - 1
    IF list[i] = target THEN
        found ← i
    ENDIF
NEXT i

// Binary search (list must be sorted)
low ← 0
high ← LENGTH(list) - 1
WHILE low <= high
    mid ← (low + high) DIV 2
    IF list[mid] = target THEN
        found ← mid
    ELSE
        IF list[mid] < target THEN
            low ← mid + 1
        ELSE
            high ← mid - 1
        ENDIF
    ENDIF
ENDWHILE
```
