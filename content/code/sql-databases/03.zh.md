# 用 WHERE 筛选行

到目前为止,每条查询都返回**所有**行。加上 `WHERE` 子句,只保留满足**条件**的行:

```sql
SELECT name, score FROM student WHERE score >= 80;
```

只有条件为真的行才会返回。

# 比较运算符

你可以这样比较:

- `=` 等于
- `<>` 不等于
- `<`  小于,`>` 大于
- `<=` 不大于,`>=` 不小于

数字直接写(`80`)。文本必须放在**单引号**里(`'11B'`):

```sql
SELECT name FROM student WHERE form = '11B';
```
