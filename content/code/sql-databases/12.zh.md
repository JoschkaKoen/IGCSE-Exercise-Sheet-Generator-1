# 两类 SQL 命令

SQL 分为两半:

- **DDL**(数据定义语言)改变**结构** —— 也就是表本身。
- **DML**(数据操纵语言)改变表里的**数据**。到目前为止学的(`SELECT`)都是 DML。

`CREATE TABLE` 是主要的 DDL 命令。它列出每一列及其**数据类型**,并标出**主键**:

```sql
CREATE TABLE customer (
  id   INTEGER PRIMARY KEY,
  name VARCHAR(20),
  city VARCHAR(15)
);
```

# 数据类型与外键

常见的数据类型:

- `INTEGER` —— 整数
- `VARCHAR(n)` / `CHAR(n)` —— 文本,最多 / 恰好 `n` 个字符
- `DECIMAL(p, s)` —— 带 `s` 位小数的数
- `DATE`、`BOOLEAN`

要连接到另一张表,加上 `FOREIGN KEY`:

```sql
CREATE TABLE orders (
  id          INTEGER PRIMARY KEY,
  customer_id INTEGER,
  FOREIGN KEY (customer_id) REFERENCES customer(id)
);
```

检查器会读回你建的表的结构,看看你创建了哪些列。
