# 用 UPDATE 修改行

`UPDATE` 修改满足 `WHERE` 条件的那些行里的值:

```sql
UPDATE product SET price = 0.95 WHERE code = 'P02';
```

`SET` 列出要修改的列。你可以一次改多列,用逗号隔开:`SET price = 0.95, in_stock = 150`。

---

# 一定要用 WHERE

`WHERE` 决定**哪些**行被修改。如果不写它,**每一**行都会被更新:

```sql
UPDATE product SET price = 0;   -- 会改掉所有产品!
```

所以几乎每条 `UPDATE` 都应该带 `WHERE`。检查器会执行你的 `UPDATE`,再读回表,确认只有该改的那一行变了。
