# 用 GROUP BY 分组

像 `AVG` 这样的聚合函数,通常把**整张**表汇总成一个数。`GROUP BY` 会把行分成共享同一个值的**组**,并为**每个组**给出一行汇总:

```sql
SELECT form, COUNT(*) AS n, ROUND(AVG(score), 2) AS avg_score
FROM student
GROUP BY form;
```

这样每个 `form` 得到一行,显示该 form 的人数和平均分。

# 用 HAVING 筛选组

`WHERE` 在分组**之前**筛选**行**。要筛选**组**本身 —— 用聚合函数 —— 就用 `HAVING`:

```sql
SELECT form, COUNT(*) AS n
FROM student
GROUP BY form
HAVING COUNT(*) >= 3;
```

这样只保留有 3 名或更多学生的 form。(`WHERE` 不能测试 `COUNT(*)`,而 `HAVING` 可以。)
