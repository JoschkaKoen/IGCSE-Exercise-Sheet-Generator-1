# 组合条件

一个 `WHERE` 子句可以同时测试不止一件事。用下面的词连接条件:

- `AND` —— **两个**都必须为真
- `OR` —— **至少一个**为真
- `NOT` —— 把条件反过来

```sql
SELECT name FROM student WHERE form = '11A' AND score > 88;
```

这会返回那些 11A 班、**并且**分数高于 88 的学生。

---

# 混用 AND 和 OR

当你混用 `AND` 和 `OR` 时,用**括号**把意思写清楚,就像在数学里一样:

```sql
SELECT name FROM student
WHERE form = '11B' OR score >= 95;
```

`AND` 会先于 `OR` 判断,所以括号能消除歧义。
