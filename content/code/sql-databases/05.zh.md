# 给结果排序

返回的行没有固定顺序。加上 `ORDER BY`,按某一列把它们排序:

```sql
SELECT name, score FROM student ORDER BY score DESC;
```

- `ASC` 表示**升序**(从小到大)—— 这是默认值。
- `DESC` 表示**降序**(从大到小)。

---

# 按多列排序

按文本列可以按字母排序:

```sql
SELECT name FROM student ORDER BY name;
```

你可以按多列排序:`ORDER BY form ASC, score DESC` 先按 form 排,再在**每个** form 内部按 score 排。`ORDER BY` 总是写在查询的最后。
