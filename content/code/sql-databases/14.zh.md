# 用 INSERT 添加行

`INSERT` 给表添加一个新行。先列出列,再给出一组对应的 `VALUES`:

```sql
INSERT INTO product (code, title, price, in_stock)
VALUES ('P04', 'Ruler', 1.20, 75);
```

文本放在单引号里,数字不用。值必须和列一一对应,顺序也要一致。

---

# 查看效果

`INSERT` 改变数据,但本身不返回任何行,所以**运行**只会报告改了多少行。要看到新行,之后运行一条 `SELECT`。

检查器会替你这么做:它先执行你的 `INSERT`,再读回这张表,确认你的行确实在里面。
