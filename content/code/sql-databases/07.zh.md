# 用 LIKE 匹配文本模式

`=` 只能**精确**匹配文本。要匹配一个**模式**,用 `LIKE` 配合两个通配符:

- `%` 代表任意一串字符(也可以是零个)
- `_` 代表恰好一个字符

```sql
SELECT name FROM student WHERE name LIKE 'S%';
```

`'S%'` 表示“以 S 开头”。`'%a%'` 表示“包含一个 a”。`'_o%'` 表示“第二个字母是 o”。

# 列表与区间:IN 和 BETWEEN

还有两个好用的测试:

- `IN (…)` 匹配列表中的任意一个值:`WHERE form IN ('11A', '11C')`
- `BETWEEN low AND high` 匹配一个区间,**含**两端:

```sql
SELECT name, score FROM student WHERE score BETWEEN 70 AND 90;
```

两者都比写好几个 `OR` 更简短。
