# 报表:连接加分组

真正的威力来自把你学过的东西组合起来:**连接**两张表,把连接后的行**分组**,再用聚合函数为每组做**汇总**。

一个常见的报表是“每位顾客花了多少钱?”—— 把顾客连接到他们的订单,按顾客分组,再 `SUM` 出金额。

---

# 每位顾客一行

```sql
SELECT customer.name,
       COUNT(*) AS orders,
       ROUND(SUM(orders.total), 2) AS spent
FROM customer
INNER JOIN orders ON customer.id = orders.customer_id
GROUP BY customer.id
ORDER BY customer.name;
```

按顺序来读:先连接两张表,再按顾客把行分组,然后为每组数出订单数、加总金额。
