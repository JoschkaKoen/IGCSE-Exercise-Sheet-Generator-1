# 数据库与表

**数据库**把信息存放在**表**里。表是一个网格,有点像电子表格:

- 每一**行**是一条**记录** —— 例如,一名学生;
- 每一**列**是一个**字段** —— 例如,学生的姓名或分数。

本课程使用一张小表,叫 `student`。它有 5 条记录、4 个字段:`id`、`name`、`form` 和 `score`。

---

# 用 SELECT 取数据

要从表里读取数据,你用 `SELECT` 写一条**查询**。

最简单的查询取**每一行**的**每一列**。星号 `*` 表示“所有列”:

```sql
SELECT * FROM student;
```

- `SELECT` 选择列 —— `*` 表示全部列。
- `FROM student` 指定读取哪张表。

数据库会返回一张**结果表**(也叫结果集):就是你的查询要的那些行。

---

# 几条规则

- 像 `SELECT` 和 `FROM` 这样的关键字常写成大写,这样更醒目。
- 每条语句以分号 `;` 结尾。
- 空单元格显示为 `NULL`,意思是“没有值”。
