# 用 DELETE 删除行

`DELETE` 删除满足 `WHERE` 条件的那些行:

```sql
DELETE FROM product WHERE code = 'P03';
```

只有匹配的行被删除;表和它其余的行都保留。

---

# 同样的警告

和 `UPDATE` 一样,不带 `WHERE` 的 `DELETE` 会作用于**每一**行:

```sql
DELETE FROM product;   -- 会清空整张表!
```

运行 `DELETE` 之前,一定要检查你的 `WHERE`。检查器会执行你的 `DELETE`,再读回表,确认该删的那一行确实没了。
