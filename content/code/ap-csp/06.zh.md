## 程序会做选择

- **选择**(selection)是指程序根据一个条件来决定做什么。
- 条件是一个结果为 `True` 或 `False` 的问题。
- 这是 AP CSP 的核心思想:会**做判断**的代码。

---

## if / elif / else

- `if` **只有**当条件为 `True` 时才运行它的代码块。
- `elif`(“否则如果”)在上面的条件都为 `False` 时再检查另一个条件。
- `else` 在**所有**条件都不为 `True` 时运行。

---

```python
hour = 14
if hour < 12:
    print("Morning")
elif hour < 18:
    print("Afternoon")
else:
    print("Evening")
```

---

## 比较运算产生条件

- `==` 等于,`!=` 不等于,`<` `>` `<=` `>=`。
- `==` **检测**两个值是否相等;单个 `=` 是**存储**一个值。
- 比较运算的结果是 `True` 或 `False`。

---

```python
score = 72
print(score >= 50)
print(score == 100)
```

---

## 布尔逻辑:and / or / not

- 只有**两边都**为 `True` 时,`and` 才为 `True`。
- 只要**至少有一边**为 `True`,`or` 就为 `True`。
- `not` 会取反:`not True` 是 `False`。

---

```python
age = 16
if age >= 13 and age <= 19:
    print("Teenager")
else:
    print("Not a teenager")
```

---

## 用 AP CSP 伪代码表示

- Python 的 `if/else` 对应 CB 的 `IF(condition){ } ELSE { }`。
- CB 用单词来表示逻辑:`AND`、`OR`、`NOT`。
- `DISPLAY` = print,`←` = `=`。

```text
IF (score >= 50)
{
    DISPLAY "Pass"
}
ELSE
{
    DISPLAY "Fail"
}

IF ((age >= 13) AND (age <= 19))
{
    DISPLAY "Teenager"
}
```
