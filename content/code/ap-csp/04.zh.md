## 变量保存值

- **变量**(variable)用来存储一个值,方便以后使用。
- 在 Python 里写作 `score = 10`。`=` 的意思是"存到左边"。
- 你可以通过把一个新值存进去来改变一个变量。

---

```python
score = 10
score = score + 5
print(score)
```

---

## 算术运算符

- 常用的运算符是 `+`、`-`、`*`、`/`。
- `//` 表示**整除**(不要余数);`**` 表示**乘方**。
- `%` 表示相除后剩下的**余数**。

---

```python
print(17 // 5)   # whole 5s that fit = 3
print(17 % 5)    # what is left over = 2
print(2 ** 3)    # 2 to the power 3 = 8
```

---

## 关系运算符

- 这些运算符**比较**两个值,给出 `True` 或 `False`。
- `==` 是"等于",`!=` 是"不等于"。
- `<`、`>`、`<=`、`>=` 比较大小。

---

```python
print(5 == 5)    # True
print(5 != 3)    # True
print(7 < 2)     # False
```

---

## 逻辑运算符

- `and` 只有在**两边**都为 `True` 时才是 `True`。
- `or` 在**至少一边**为 `True` 时就是 `True`。
- `not` 把 `True` 变成 `False`,反过来也一样。

---

```python
print(True and False)   # False
print(True or False)    # True
print(not True)         # False
```
