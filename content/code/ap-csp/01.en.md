## Variables

- A **variable** stores a value so you can use it later.
- In Python you write `score = 10`.
- AP pseudocode writes the same idea as `score ← 10`.

---

```python
score = 10
score = score + 5
print(score)
```

---

## Arithmetic and MOD

- The usual operators are `+`, `-`, `*`, `/`.
- `//` gives **whole-number** division (no remainder).
- `%` gives the **remainder**. AP pseudocode calls this **`MOD`**.

---

```python
print(17 // 5)   # how many whole 5s fit
print(17 % 5)    # what is left over (MOD)
```
