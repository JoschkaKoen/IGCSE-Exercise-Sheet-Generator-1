## 用迭代来重复

- **迭代**(iteration)是指一遍又一遍地做同样的步骤。
- 循环让你不用把同一行代码写很多次。
- AP CSP 重视循环,因为它们让一点点代码就能完成大量工作。

---

## 用 for 和 range 计数

- `for i in range(n):` 把循环体重复 `n` 次。
- `range(5)` 给出 `0, 1, 2, 3, 4`(在 5 **之前**停止)。
- 循环体要**缩进**,就像 `if` 一样。

---

```python
for i in range(5):
    print(i)
```

---

## 累加出一个总和(accumulation)

- **累加器**(accumulator)是在循环里不断变大的变量。
- 在循环**之前**把它从 `0` 开始。
- 每一轮往里加,累出一个运行中的总和。

---

```python
total = 0
for n in range(1, 6):
    total = total + n
print(total)
```

---

## 用 while 循环直到某个条件

- `while` 循环**只要**条件为 `True` 就一直重复。
- 循环里必须有东西朝着让条件变成 `False` 的方向走。
- 当你事先不知道次数时使用它。

---

```python
count = 3
while count > 0:
    print(count)
    count = count - 1
```

---

## 遍历一个列表(for-each)

- `for item in mylist:` 依次访问每个元素。
- 循环变量每一轮保存一个元素。
- 这是处理每个值的简洁方法。

---

```python
scores = [10, 20, 30]
for s in scores:
    print(s)
```

---

## 用 AP CSP 伪代码表示

- CB 有三种循环形式,与 Python 的循环相对应。
- `REPEAT n TIMES` = 固定次数;`REPEAT UNTIL(cond)` = 循环直到条件为真。
- `FOR EACH x IN list` = 逐项循环。`DISPLAY` = print,`←` = `=`,`MOD` = `%`。

```text
REPEAT 5 TIMES
{
    DISPLAY "hi"
}

count ← 3
REPEAT UNTIL (count = 0)
{
    DISPLAY count
    count ← count - 1
}

FOR EACH s IN scores
{
    DISPLAY s
}
```
