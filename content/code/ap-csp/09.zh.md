## 我们要做什么

- **算法**(algorithm)是一份解决问题的清晰步骤清单。
- 一个很常见的任务是**搜索**:找出某个值在列表中的位置。
- 我们要学两个算法:**线性搜索**和**二分搜索**。

---

## 线性搜索

- **一个一个**地检查元素,从头到尾。
- 如果找到了这个值,就返回它的**索引**(位置)。
- 如果一直到末尾都没找到,就返回 `-1`。

---

```python
def linear_search(lst, target):
    for i in range(len(lst)):
        if lst[i] == target:
            return i
    return -1

print(linear_search([4, 8, 15, 16], 15))
print(linear_search([4, 8, 15, 16], 99))
```

---

## 为什么有序列表有帮助

- 线性搜索对**任何**列表都有效,哪怕是乱序的。
- 但如果列表是**有序的**(从小到大),我们可以快得多。
- **二分搜索**利用有序的顺序,每次跳过一半的列表。

---

## 二分搜索

- 看**中间**的元素。
- 如果它就是目标,你就完成了。
- 如果目标**更小**,就搜索**左**半边;如果**更大**,就搜索**右**半边。重复这个过程。

---

```python
def binary_search(sorted_lst, target):
    low = 0
    high = len(sorted_lst) - 1
    while low <= high:
        mid = (low + high) // 2
        if sorted_lst[mid] == target:
            return mid
        elif sorted_lst[mid] < target:
            low = mid + 1
        else:
            high = mid - 1
    return -1

print(binary_search([1, 3, 5, 7, 9], 7))
print(binary_search([1, 3, 5, 7, 9], 4))
```

---

## 把这些想法组合起来

- 一次搜索**组合**了三个你已经学过的基本部件。
- **顺序**:按次序执行步骤。**选择**:`if` / `elif` / `else`。
- **迭代**:用循环(`for` 或 `while`)重复检查。

---

## 用 AP CSP 伪代码表示

- 考试会像这样写一个循环和一个检查。
- `FOR EACH` 访问每个元素;`IF` 做选择;`REPEAT UNTIL` 一直循环到某个条件为真。

```text
PROCEDURE contains(list, target)
{
    FOR EACH item IN list
    {
        IF (item = target)
        {
            RETURN(true)
        }
    }
    RETURN(false)
}
```
