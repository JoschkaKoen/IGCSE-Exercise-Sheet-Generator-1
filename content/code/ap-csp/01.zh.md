---

## 变量

- **变量**(variable)用来存储一个值,方便以后使用。
- 在 Python 里写作 `score = 10`。
- AP 伪代码用 `score ← 10` 表示同样的意思。

---

```python
score = 10
score = score + 5
print(score)
```

---

## 算术运算与 MOD

- 常用的运算符是 `+`、`-`、`*`、`/`。
- `//` 表示**整除**(只取整数部分,不要余数)。
- `%` 表示**取余**(remainder)。AP 伪代码把它叫作 **`MOD`**。

---

```python
print(17 // 5)   # 能放下几个完整的 5
print(17 % 5)    # 剩下多少(MOD)
```
