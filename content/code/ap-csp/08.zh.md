## 我们要做什么

- **过程**(procedure)是一段有名字的代码,你可以反复使用它。
- 在 Python 里,过程用 `def` 来创建。
- 这是大概念 3 的一部分:用可复用的部件来搭建程序。

---

## 创建一个过程

- 先写 `def`,然后是**名字**,再加 `()`。
- 把代码**缩进**写在下面。
- 写出过程的名字加 `()` 就可以调用它。

---

```python
def greet():
    print("Hello!")

greet()
greet()
```

---

## 参数把数据传进来

- **参数**(parameter)是过程接收到的一个值。
- 把参数名字写在 `()` 里面。
- 在过程内部,你像使用变量一样使用它。

---

```python
def greet(name):
    print("Hello, " + name)

greet("Mia")
greet("Sam")
```

---

## `return` 把值送回来

- `return` 把一个值**送出**过程。
- 调用者可以存储这个值或者使用它。
- 执行到 `return` 之后,过程就停止。

---

```python
def square(n):
    return n * n

answer = square(5)
print(answer)
```

---

## 为什么过程有用:抽象

- **抽象**(abstraction)是指把细节藏在一个简单的名字后面。
- 你调用 `square(5)`,不用去想它内部是怎么运作的。
- 这让代码更短、更清楚,也容易**复用**。

---

## 用 AP CSP 伪代码表示

- 考试使用一种伪代码来表示过程。
- `PROCEDURE` 就像 `def`;`RETURN` 就像 `return`。

```text
PROCEDURE square(n)
{
    RETURN(n * n)
}
```
