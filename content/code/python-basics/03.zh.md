## 两种数字

- **整数**(`int`)是没有小数的数,例如 `7` 或 `-3`。
- **浮点数**(`float`)是带小数点的数,例如 `2.5`。
- 用 `/` 做除法总是得到浮点数,连 `4 / 2` 也会得到 `2.0`。

---

## 加、减、乘、除

- `+` 加,`-` 减,`*` 乘,`/` 除。
- 你可以对变量使用它们,也可以直接对数字使用。
- 运算符两边的空格可有可无,但加上会让代码更易读。

---

```python
print(7 + 2)
print(7 - 2)
print(7 * 2)
print(7 / 2)
```

---

## 整除、求余、乘方

- `//` 做除法,只保留**整数**部分(向下取整,floor division)。
- `%` 求**余数**(叫作取模,modulo)。
- `**` 表示**乘方**:`2 ** 3` 就是 2×2×2。

---

```python
print(7 // 2)
print(7 % 2)
print(2 ** 5)
```

---

## 运算顺序

- Python 先算 `**`,再算 `*` `/` `//` `%`,最后算 `+` `-`。
- 这和你在数学里用的顺序一样。
- 用**括号** `(...)` 来改变运算顺序。

---

```python
print(2 + 3 * 4)
print((2 + 3) * 4)
```
