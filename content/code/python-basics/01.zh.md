---

## 用 `print` 显示文字

- `print(...)` 会把文字显示在屏幕上。
- 把文字放进引号里:`"..."`。
- 每个 `print` 都会另起**新的一行**。

---

```python
print("Welcome to Python!")
print("This line runs next.")
```

---

## 字符串

- 引号里的文字叫作**字符串**(string)。
- 你可以用双引号 `"..."`,也可以用单引号 `'...'`。
- 不加引号也可以打印一个**数字**,例如 `print(42)`。

---

## 注释

- 以 `#` 开头的一行是**注释**(comment)。
- Python 会**忽略**注释 —— 它们不会运行。
- 用注释给阅读代码的人留下说明。

---

```python
# This line is a note for people. Python skips it.
print("Comments do not show.")
```
