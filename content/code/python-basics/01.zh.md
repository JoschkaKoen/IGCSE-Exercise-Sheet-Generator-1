## 用 `print` 显示文字

- `print(...)` 会把文字显示在屏幕上。
- 把文字放进引号里:`"..."`。这叫作**字符串**。

---

```python
print("Hello!")
```

---

## 用 `input` 向用户提问

- `input()` 会等待用户输入一行文字。
- 把它存进一个**变量** —— 一个有名字的盒子,稍后可以使用。
- 用 `+` 把两个字符串连接起来。

---

```python
name = input()
print("Hello, " + name)
```

---

## 编写函数

- **函数**是一段有名字、可以反复使用的代码。
- 用 `def` 定义它,用 `return` 返回一个值。
- 函数内部的行要**缩进**(4 个空格)。

---

```python
def double(x):
    return x * 2

print(double(5))
```
