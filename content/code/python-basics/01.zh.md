## 欢迎来到 Python

Python 是一门编程语言。你写下**指令**,计算机就会从上到下、一行一行地执行它们。

在这节课里,你将直接在浏览器中运行自己的代码。无需安装任何东西。把代码输入到框里,然后按**运行**。

## 用 `print` 显示文字

最常用的指令是 `print`,它会把文字显示在屏幕上。

```
print("Hello!")
```

引号 `"..."` 里的文字叫作**字符串**,也就是一段文本。你放在引号之间的内容就是会被显示出来的内容。先运行这个例子,再改改里面的文字,然后再运行一次。

现在来做**任务 1**:让 Python 输出 `Hello, world!`。

## 用 `input` 向用户提问

程序也可以向用户索取信息。`input()` 指令会等待用户输入一行文字,并把这段文字交还给你。我们通常把它存进一个**变量**,以便稍后使用:

```
name = input()
print(name)
```

变量就像一个贴了标签、用来存放数据的盒子。上面第一行把用户输入的内容放进名为 `name` 的盒子里,第二行再把它打印出来。

要把几段文字拼接在一起,在它们之间放一个 `+`:

```
print("Hi, " + name + "!")
```

## 用函数重复使用代码

**函数**是一段有名字的代码,可以反复使用。用 `def` 定义函数,用 `return` 把一个值返回出去:

```
def double(x):
    return x * 2
```

这个函数叫 `double`。它接收一个输入 `x`,并返回 `x * 2`。注意函数内部的那一行是**缩进**的(用四个空格往右推)。缩进是 Python 判断哪些行属于这个函数的方式。
