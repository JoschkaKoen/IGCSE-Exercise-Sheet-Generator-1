## 按名字存放数据

- **字典**(dictionary)把值存放在叫作**键**(key)的名字下。
- 把键值对写在花括号里:`{"name": "Sam", "age": 15}`。
- 每个键都是唯一的,指向一个值。

---

## 读取和设置值

- 用键来读取值:`person["name"]`。
- 用同样的方式添加或修改值:`person["age"] = 16`。
- 列表用数字索引;字典用键。

---

```python
ages = {"Sam": 12, "Mia": 15}
print(ages["Sam"])
ages["Leo"] = 9
print(ages)
```

---

## 用字典表示记录

- **记录**(record)把关于某一事物的相关信息组合在一起。
- 字典是保存一条记录的简便方式。
- 每个键就是一个**字段**(field),例如 `name` 或 `grade`。

---

```python
pupil = {"name": "Ada", "grade": "A"}
print(pupil["name"])
print(pupil["grade"])
```

---

## 判断和遍历

- 如果某个键存在,`"name" in person` 就是 `True`。
- `for key in person:` 会依次访问每个键。
- 如果键不存在,`person.get("x", 0)` 会返回一个默认值。

---

```python
menu = {"tea": 2, "cake": 5}
for item in menu:
    print(item, menu[item])
print("tea" in menu)
```

---

## 用剑桥伪代码表示

- 字典对应考试里的 `TYPE` 记录;每个键就是一个字段,用点号来访问。

```text
TYPE Student
    DECLARE Name : STRING
    DECLARE Age  : INTEGER
ENDTYPE

DECLARE pupil : Student
pupil.Name ← "Sam"
pupil.Age  ← 15
```
