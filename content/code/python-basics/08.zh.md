## 当条件为真时循环

- `while` 循环会**只要**条件为真就一直重复。
- 它在每一轮**之前**检查条件。
- 当你事先不知道要重复多少次时,就用它。

---

## 一定要改变条件

- 循环里必须有某些东西逐渐让条件变为假。
- 如果条件永远不变假,循环就会一直运行(**死循环**,infinite loop)。
- 如果循环卡住了,按**停止**。

---

```python
count = 3
while count > 0:
    print(count)
    count = count - 1
```

---

## break 和 continue

- `break` 会立刻停止循环。
- `continue` 会跳过本轮剩下的部分,直接进入下一轮。
- `while True:` 会一直循环,直到某个 `break` 结束它。

---

```python
n = 0
while True:
    n = n + 1
    if n == 4:
        break
    print(n)
```

---

## 先检查还是后检查

- `while` 循环**先**检查,所以它可能运行**零**次。
- "重复直到"型循环**后**检查,所以它至少运行**一**次。
- Python 没有 `repeat`;可以用 `while True:` 并在末尾加一个 `break`。

---

## 用剑桥伪代码表示

- `while` 是前置条件循环;`REPEAT … UNTIL` 是后置条件循环。

```text
count ← 3
WHILE count > 0
    OUTPUT count
    count ← count - 1
ENDWHILE

REPEAT
    OUTPUT count
    count ← count + 1
UNTIL count = 3
```
