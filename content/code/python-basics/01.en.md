## Show text with `print`

- `print(...)` shows text on the screen.
- Put the text in quotes: `"..."`. This is called a **string**.

---

```python
print("Hello!")
```

---

## Ask the user with `input`

- `input()` waits for the user to type a line.
- Save it in a **variable** — a named box you can use later.
- Join two strings with `+`.

---

```python
name = input()
print("Hello, " + name)
```

---

## Make a function

- A **function** is named code you can use again.
- `def` makes it. `return` gives a value back.
- The lines inside are **indented** (4 spaces).

---

```python
def double(x):
    return x * 2

print(double(5))
```
