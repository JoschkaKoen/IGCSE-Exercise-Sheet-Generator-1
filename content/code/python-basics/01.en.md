## Show text with `print`

- `print(...)` shows text on the screen.
- Put the text in quotes: `"..."`.
- Each `print` starts a **new line**.

---

```python
print("Welcome to Python!")
print("This line runs next.")
```

---

## Strings

- Text in quotes is called a **string**.
- You can use double quotes `"..."` or single quotes `'...'`.
- You can also print a **number** with no quotes, like `print(42)`.

---

## Comments

- A line that starts with `#` is a **comment**.
- Python **ignores** comments — they do not run.
- Use them to leave notes for people who read your code.

---

```python
# This line is a note for people. Python skips it.
print("Comments do not show.")
```
