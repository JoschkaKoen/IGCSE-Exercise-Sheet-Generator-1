## Welcome to Python

Python is a programming language. You write **instructions**, and the computer
follows them one line at a time, from top to bottom.

In this lesson you will run your own code right here in the browser. There is
nothing to install. Type code into the box and press **Run**.

## Showing text with `print`

The most common instruction is `print`. It shows text on the screen.

```
print("Hello!")
```

The words inside the quotes `"..."` are called a **string** — just a piece of
text. Whatever you put between the quotes is what gets shown. Run the example,
then change the words and run it again.

Now try **Task 1**: make Python print `Hello, world!`.

## Asking the user with `input`

A program can also ask the user for something. The `input()` instruction waits
for the user to type a line and gives that text back to you. We usually save it
in a **variable** so we can use it later:

```
name = input()
print(name)
```

A variable is a labelled box that holds a value. Here the first line puts
whatever the user typed into a box called `name`, and the second line prints it.

To join pieces of text together, put a `+` between them:

```
print("Hi, " + name + "!")
```

## Reusing code with functions

A **function** is a named piece of code you can use again and again. You define
one with `def`, and `return` sends a value back:

```
def double(x):
    return x * 2
```

This function is called `double`. It takes one input, `x`, and returns `x * 2`.
Notice the line inside the function is **indented** (pushed to the right with
four spaces). Indentation is how Python knows which lines belong to the
function.
