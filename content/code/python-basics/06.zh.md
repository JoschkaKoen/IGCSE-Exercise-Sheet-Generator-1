## 做出选择

- `if` **只有当**条件为真时才会运行某段代码。
- 条件后面要加一个冒号 `:`。
- 属于这个 `if` 的代码行要**缩进**(4 个空格)。

---

## 比较

- 条件通常是一个为真或为假的比较。
- `==` 等于,`!=` 不等于,`<` `>` `<=` `>=`。
- 注意:`==` 用来**判断**是否相等;单个 `=` 用来**存值**。

---

```python
temperature = 30
if temperature > 25:
    print("It is hot.")
```

---

## if / elif / else

- 如果第一个条件为假,`elif`("else if")会再检查另一个条件。
- 当**没有**任何条件为真时,`else` 才会运行。
- Python 会**按顺序**检查,遇到第一个为真的就停下。

---

```python
hour = 14
if hour < 12:
    print("Morning")
elif hour < 18:
    print("Afternoon")
else:
    print("Evening")
```

---

## 组合条件

- `and`:只有**两边都**为真时才为真。
- `or`:**至少有一边**为真时就为真。
- `not`:把真变假,把假变真。

---

```python
day = "Sat"
if day == "Sat" or day == "Sun":
    print("Weekend!")
```

---

## 用剑桥伪代码表示

- Python 的 `if/elif/else` 对应考试里的 `IF … ELSE … ENDIF` 和 `CASE`。

```text
IF score >= 50 THEN
    OUTPUT "Pass"
ELSE
    OUTPUT "Fail"
ENDIF

CASE OF grade
    "A": OUTPUT "Top"
    "B": OUTPUT "Good"
    OTHERWISE: OUTPUT "Keep trying"
ENDCASE
```
