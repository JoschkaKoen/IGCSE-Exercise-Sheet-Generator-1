## 有名字的盒子

- **变量**(variable)是一个有名字的盒子,用来存放一个值。
- 你把值放进盒子,稍后用名字来使用它。
- 这个值可以是数字,也可以是字符串。

---

## 创建变量

- 写 `name = value` 来存放一个值。
- **名字**在左边,**值**在右边。
- `=` 的意思是"把这个值存进去",而不是"等于"。

---

```python
city = "Paris"
print(city)
```

---

## 命名规则

- 可以用字母、数字和下划线 `_`。
- 名字不能以数字开头,也不能有空格。
- 名字**区分大小写**:`score` 和 `Score` 是不同的。
- 选清楚的名字,例如 `age` 或 `total_score`。

---

## 修改变量

- 你可以随时存入一个新的值。
- 新的值会**替换**旧的值。
- 名字始终指向一个值 —— 最新的那个。

---

```python
count = 1
count = 2
print(count)
```

---

## 常量

- **常量**(constant)是一个不应该改变的值。
- Python 没有专门的关键字来表示它。
- 按照**惯例**,我们把常量的名字写成 `UPPERCASE`(全大写)。

---

```python
PI = 3.14159
print(PI)
```
