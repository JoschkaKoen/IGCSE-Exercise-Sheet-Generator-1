## 用 `input` 向用户提问

- `input()` 会等待用户输入一行文字并按回车。
- 它会把用户输入的内容返回。把它存进一个变量。
- 用 `+` 把两个字符串连接起来。

---

```python
name = input()
print("Welcome, " + name)
```

---

## input 永远返回字符串

- 不管用户输入什么,返回的都是**字符串**。
- 即使他们输入 `5`,你得到的也是文字 `"5"`,而不是数字。
- 所以 `"5" + "5"` 是 `"55"`(把文字拼起来),而不是 `10`。

---

## 把文字变成数字

- `int(...)` 把文字变成整数:`int("5")` → `5`。
- `float(...)` 把文字变成小数:`float("2.5")` → `2.5`。
- 先转换,然后才能做数学运算。

---

```python
age = int(input())
print(age + 1)
```
