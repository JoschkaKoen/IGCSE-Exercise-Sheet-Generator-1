## 用 `for` 重复

- `for` 循环会把代码重复固定的次数。
- 循环体要**缩进**,就像 `if` 一样。
- **循环变量**在每一轮都会取一个新的值。

---

## 用 range 计数

- `range(5)` 给出数字 `0, 1, 2, 3, 4`(到 5 之前停下)。
- `range(1, 6)` 给出 `1, 2, 3, 4, 5`(包含起点,不包含终点)。
- `range(0, 10, 2)` 以 2 为**步长**计数。

---

```python
for i in range(5):
    print(i)
```

---

## 累加求和(累加器)

- **累加器**(accumulator)是一个在循环里不断增长的变量。
- 在循环**之前**先把它设为 `0`。
- 每一轮都往里面加,从而得到累计总和。

---

```python
total = 0
for n in range(1, 6):
    total = total + n
print(total)
```

---

## 用剑桥伪代码表示

- Python 的 `for … range` 循环就是考试里的计数循环 `FOR … NEXT`。

```text
total ← 0
FOR i ← 1 TO 5
    total ← total + i
NEXT i
OUTPUT total
```
