## 把所有内容综合起来

- 你现在已经会变量、输入、`if`、循环、列表、函数和字典了。
- 一个真正的程序会把这些小部件组合成一个有用的工具。
- 这节课你将一次写一个函数,搭出一个小小的**班级成绩分析器**。

---

## 计划

- 我们有一个成绩**列表**,比如 `[70, 90, 50, 80]`。
- 我们想要三个答案:**平均分**、有多少人**及格**,以及一段简短的**总结**。
- 我们会为每件事写一个小**函数**,然后让它们协同工作。

---

## 处理列表的便捷内置函数

- Python 提供了一些现成的小助手,可以直接用在数字列表上。
- `len(x)` 数有多少个,`sum(x)` 求和,`max(x)` / `min(x)` 找出最大 / 最小值。
- 用它们,而不必每次都自己写一个循环。

---

```python
scores = [70, 90, 50, 80]
print(len(scores))   # 4
print(sum(scores))   # 290
print(max(scores))   # 90
print(min(scores))   # 50
```

---

## 一个你会反复用到的套路:统计通过测试的元素

- 一个很常见的任务:遍历一个列表,**统计**有多少元素通过某个测试。
- 让计数器从 `0` 开始,每当 `if` 为真时就加 `1`。
- 这里我们统计长单词;很快你就会去统计及格的成绩。

---

```python
def how_many_long_words(words, min_length):
    count = 0
    for w in words:
        if len(w) >= min_length:
            count = count + 1
    return count

print(how_many_long_words(["hi", "hello", "hey", "howdy"], 4))   # 2
```
