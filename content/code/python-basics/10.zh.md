## 给一段代码起个名字

- **函数**(function)是一段有名字、可以反复使用的代码。
- 用 `def name():` 加上缩进的函数体来定义它。
- 用名字"调用"这个函数,就会运行那段代码。

---

## 给它输入(参数)

- 括号里的名字是**参数**(parameter)—— 函数的输入。
- 调用它时,你传入**实参**(argument,真正的值)。
- 一个函数可以有多个参数。

---

```python
def shout(word):
    print(word.upper() + "!")

shout("hello")
```

---

## 返回一个结果(return)

- `return` 会把一个值**返回**给调用它的地方。
- 你可以存下这个值,或在表达式里使用它。
- `return` 还会立刻结束这个函数。

---

```python
def double(x):
    return x * 2

result = double(5)
print(result)
```

---

## return 与 print 的区别

- `print` 只是把内容**显示**在屏幕上。
- `return` 会**返回一个值**,让后面的代码可以使用。
- 没有 `return` 的函数会返回 `None`。

---

## 用剑桥伪代码表示

- `FUNCTION` 会返回一个值;`PROCEDURE` 只是执行一个动作。

```text
FUNCTION Add(a : INTEGER, b : INTEGER) RETURNS INTEGER
    RETURN a + b
ENDFUNCTION

PROCEDURE Greet(name : STRING)
    OUTPUT "Hello, ", name
ENDPROCEDURE
```

---

## 现在轮到你

- 定义好每个函数,检查会替你调用它。
- 用 `return` 把答案返回(不要用 `print` 打印)。
