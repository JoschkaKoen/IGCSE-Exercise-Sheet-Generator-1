## 局部变量

- 在函数**内部**创建的变量是**局部的**(local)。
- 它只在函数运行期间存在。
- 函数外面的代码看不到它。

---

```python
def make_message():
    note = "secret"
    print(note)

make_message()
# print(note)  would be an error: note is local
```

---

## 全局变量

- 在所有函数**外部**创建的变量是**全局的**(global)。
- 函数可以**读取**全局值。
- 想在函数里**修改**全局变量,需要用 `global` 关键字 —— 但通常更清晰的做法是用 `return` 返回一个值。

---

## 默认值

- 参数可以有一个**默认值**:`def f(x, step=1):`。
- 如果调用时没给这个参数,就使用默认值。
- 带默认值的参数要放在普通参数**后面**。

---

```python
def power(base, exp=2):
    return base ** exp

print(power(5))
print(power(2, 3))
```

---

## 实参是怎样传递的

- Python 传递的是指向对象的**引用**(reference)。
- 数字和字符串是**不可变的**(immutable):函数无法改变调用者的值。
- 列表是**可变的**(mutable):在函数内部所做的修改**确实会**影响调用者的列表。

---

```python
def try_change(n):
    n = n + 100   # changes only the local copy
    print(n)

x = 5
try_change(x)
print(x)          # x is still 5
```

---

## 用剑桥伪代码表示

- `BYVAL` 传递一个副本(调用者不变);`BYREF` 共享同一份数据(调用者会改变)。

```text
PROCEDURE Increase(BYREF n : INTEGER)
    n ← n + 1
ENDPROCEDURE

PROCEDURE Show(BYVAL n : INTEGER)
    OUTPUT n
ENDPROCEDURE
```
