## 列表能存放很多值

- **列表**(list)在一个变量里存放很多值。
- 把这些项写在方括号里:`[10, 20, 30]`。
- 这些项可以是数字、字符串,或两者混合。

---

## 获取和修改元素

- 用索引,就像字符串一样:`nums[0]` 是第一个元素。
- `nums[-1]` 是最后一个元素。
- 你也可以**修改**某个元素:`nums[0] = 99`。

---

```python
fruits = ["apple", "pear", "plum"]
print(fruits[0])
print(fruits[-1])
```

---

## 添加元素并计数

- `nums.append(x)` 会把 `x` 加到列表的**末尾**。
- `len(nums)` 会告诉你有多少个元素。
- 列表可以在程序运行时不断增长。

---

```python
fruits = ["apple", "pear"]
fruits.append("plum")
print(len(fruits))
print(fruits)
```

---

## 遍历列表

- `for item in nums:` 会依次访问每个元素。
- 循环变量每一轮持有一个元素。
- 这是处理每个值的简便方法。

---

```python
scores = [10, 20, 30]
total = 0
for s in scores:
    total = total + s
print(total)
```

---

## 用剑桥伪代码表示

- 列表就是考试里的**一维数组**(1-D array)。注意:考试里的数组常常从索引 `1` 开始,而 Python 列表从 `0` 开始。

```text
DECLARE scores : ARRAY[1:3] OF INTEGER
scores[1] ← 10
scores[2] ← 20
scores[3] ← 30
FOR i ← 1 TO 3
    OUTPUT scores[i]
NEXT i
```
