## Ask the user with `input`

- `input()` waits for the user to type a line and press Enter.
- It gives back what they typed. Store it in a variable.
- Join two strings together with `+`.

---

```python
name = input()
print("Welcome, " + name)
```

---

## input always gives a string

- Whatever the user types comes back as a **string**.
- Even if they type `5`, you get the text `"5"`, not the number.
- So `"5" + "5"` is `"55"` (joined text), not `10`.

---

```python
print("5" + "5")   # 55 — joined text, not the number 10
```

---

## Turn text into a number

- `int(...)` turns text into a whole number: `int("5")` → `5`.
- `float(...)` turns text into a decimal: `float("2.5")` → `2.5`.
- Convert first, then you can do maths.

---

```python
age = int(input())
print(age + 1)
```
