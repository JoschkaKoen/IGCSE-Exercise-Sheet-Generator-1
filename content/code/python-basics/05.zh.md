## 字母有位置

- 字符串是一串**字符**(character)。
- 每个字符都有一个位置,叫作**索引**(index)。
- 索引从 `0` 开始。用 `s[-1]` 表示**最后**一个字符。

---

```python
word = "Python"
print(word[0])
print(word[-1])
```

---

## 取一段切片

- **切片**(slice)从字符串中取出一部分:`s[start:end]`。
- 位置 `end` 处的字符**不**包含在内。
- 留空一侧表示"从头开始"或"到末尾":`s[:3]`、`s[2:]`。

---

```python
msg = "hello"
print(msg[:3])
print(msg[2:])
```

---

## 长度与大小写

- `len(s)` 返回字符的个数。
- `s.upper()` 生成一个全大写的副本;`s.lower()` 生成一个全小写的副本。
- 它们会生成**新的**字符串;原来的字符串不会改变。

---

```python
city = "Tokyo"
print(len(city))
print(city.upper())
print(city.lower())
```

---

## 用 f-string 拼接文字

- **f-string** 在引号前加一个 `f`。
- 在 `{ }` 里放一个值,Python 会把它填进文字中。
- 这是把文字和值混在一起的简洁写法。

---

```python
fruit = "apple"
count = 3
print(f"I have {count} {fruit}s")
```
