## 使用对象

- **对象**(object)是一个会做事情的值。你用点号 `.` 来调用它的**方法**(method)。
- `String` 就是一个对象。`Math`、`Integer` 和 `Double` 给你现成的工具。
- 在这节课里,你学习如何**创建**对象并**使用**它们的方法。

---

## 用 `new` 创建对象

- 有些对象用关键字 **`new`** 和一个**构造器**(constructor)来创建。
- `String` 很特别:你可以直接用引号写它,例如 `"hi"`。
- 你也可以用 `new String(...)` 从字符创建一个 `String`,但用引号是常见的写法。

---

```java
public class Main {
    public static void main(String[] args) {
        String greeting = "Hello";
        String copy = new String("Hello");
        System.out.println(greeting);
        System.out.println(copy);
    }
}
```

---

## String 的方法

- `s.length()` —— `s` 里有多少个字符。
- `s.charAt(i)` —— 位置 `i` 上的字符(第一个位置是 **0**)。
- `s.substring(a, b)` —— 从 `a` 开始到 `b`(**不包括** `b`)的那一部分。
- `s.indexOf("x")` —— `"x"` 第一次出现的位置,找不到就返回 `-1`。

---

```java
public class Main {
    public static void main(String[] args) {
        String word = "computer";
        System.out.println(word.length());        // 8
        System.out.println(word.charAt(0));        // c
        System.out.println(word.substring(0, 4));  // comp
        System.out.println(word.indexOf("put"));   // 3
    }
}
```

## 比较字符串

- 用 `s.equals(other)` 来检查两个字符串是否有**相同的字母**。
- **不要**用 `==` 来比较字符串 —— 它可能给出错误的答案。我们在第 3 课学习原因。
- `s.compareTo(other)` 相等时返回 `0`,如果 `s` 在顺序上更靠前就返回**负数**,更靠后就返回**正数**。

---

```java
public class Main {
    public static void main(String[] args) {
        String a = "cat";
        String b = "cat";
        String c = "dog";
        System.out.println(a.equals(b));   // true
        System.out.println(a.equals(c));   // false
        System.out.println(a.compareTo(c)); // negative: "cat" is before "dog"
    }
}
```

## Math 的方法

- `Math` 装着数学工具。你在 `Math` 本身上调用它们,例如 `Math.abs(-5)`。
- `Math.abs(x)` —— `x` 去掉负号后的大小。
- `Math.pow(b, e)` —— `b` 的 `e` 次方(返回一个 `double`)。
- `Math.sqrt(x)` —— `x` 的平方根(返回一个 `double`)。
- `Math.random()` —— 一个从 `0.0` 到 `1.0`(不包括 `1.0`)的随机 `double`。

---

```java
public class Main {
    public static void main(String[] args) {
        System.out.println(Math.abs(-7));       // 7
        System.out.println(Math.pow(2, 3));     // 8.0
        System.out.println(Math.sqrt(9));       // 3.0
        // Math.random() changes each run, so we do not print it here.
    }
}
```

## 包装类与自动装箱

- `int` 和 `double` 是简单类型。它们的**对象**版本是 `Integer` 和 `Double`。
- Java 可以替你在两者之间转换。这叫作**自动装箱**(autoboxing)。
- `Integer.parseInt("42")` 把文字变成 `int`。`Double.parseDouble("3.5")` 把文字变成 `double`。

---

```java
public class Main {
    public static void main(String[] args) {
        Integer boxed = 10;        // int -> Integer (autoboxing)
        int back = boxed;          // Integer -> int (unboxing)
        int n = Integer.parseInt("42");
        double d = Double.parseDouble("3.5");
        System.out.println(back + n);   // 52
        System.out.println(d);          // 3.5
    }
}
```
