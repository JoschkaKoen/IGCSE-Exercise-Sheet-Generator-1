## 自由作答部分

- AP CSA 考试有两个部分。第一部分是选择题。第二部分是**自由作答**(free response,你手写 Java)。
- 这部分总是有 **4 道自由作答题(FRQ)**,而且每年都是**同样的 4 种类型**。
- 这一课是这 4 种类型的快速地图。了解套路会让考试不再可怕。你能做到的!

---

## 类型 1:方法与控制结构

- 你要写**一到两个方法**,用到循环、`if`,以及对 `int`、`double`、`boolean`、`String` 的简单运算。
- 不用设计类 —— 类已经给好了。你只要填上方法体。
- 这是最"纯写代码"的题目。第 1–8 课覆盖了你需要的一切。

---

```java
public class Grades {
    // Count the scores that are 60 or more.
    public static int countPassing(int[] scores) {
        int count = 0;
        for (int s : scores) {
            if (s >= 60) {
                count = count + 1;
            }
        }
        return count;
    }

    public static void main(String[] args) {
        int[] scores = {72, 40, 95, 60, 33};
        System.out.println(countPassing(scores));   // 3
    }
}
```

## 类型 2:类的设计

- 题目给你一段类的**描述**,你必须**写出整个类**。
- 也就是说:实例变量、一个**构造方法**(constructor),以及他们要求的那些方法。
- 记住用 `private` 字段、构造方法要和**类同名**、以及 `this.field = field`。第 6 课覆盖了这些。

```text
public class Student {
    private String name;
    private int score;

    public Student(String name, int score) {
        this.name = name;
        this.score = score;
    }

    public int getScore() {
        return score;
    }
}
```

## 类型 3:数组 / ArrayList

- 你要处理一个 `ArrayList`(有时是数组),并写出**遍历**(traverse)它的代码。
- 常见任务:找一个值、数个数、删除元素,或者建一个新列表。
- 在循环里**删除**元素时要小心下标。第 9–11 课覆盖了这些。

```text
// Count the Strings longer than 3 letters.
int count = 0;
for (int i = 0; i < words.size(); i++) {
    if (words.get(i).length() > 3) {
        count++;
    }
}
return count;
```

## 类型 4:二维数组

- 你要处理一个**二维数组**(一个由行和列组成的网格)。
- 你要用**嵌套循环**:外层循环走行,内层循环走列。
- 常见任务:对一行求和、对一列求和、找最大值,或者数匹配的个数。第 12 课覆盖了这些。

```text
// Add up every value in the grid.
int total = 0;
for (int r = 0; r < grid.length; r++) {
    for (int c = 0; c < grid[r].length; c++) {
        total += grid[r][c];
    }
}
return total;
```

## 考试策略:仔细读题

- **先把整道题读完。** 划出方法必须**返回**什么,以及它的**参数**是什么。
- 看他们给的**例子表格**。它准确地告诉你代码应该产生什么结果。
- 抄下他们给你的**方法头**(method header)。不要改它的名字、参数或返回类型。

---

## 考试策略:写出能用的方法

- 先写出**循环的框架**,再填里面的内容。在脑子里一步一步把它跑通。
- **正确地遍历**:下标从 `0` 开始,在 `length`(或 `size()`)处停止;检查两端的边界。
- 用 `return` 返回正确的**类型**。如果方法头写的是 `int`,就返回一个 `int`。

---

## 考试策略:小分也能累积

- 即使整个方法不完美,能用的**部分**也会得分。一定要写出*一些*东西。
- 只用这门课里你学过的 Java。简单、正确的代码胜过花哨的代码。
- 继续下一题之前,用题目里的例子数值检查一下你的答案。
