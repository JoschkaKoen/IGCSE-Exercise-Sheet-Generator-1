## 布尔值与选择

- **布尔值**(boolean)是一个值,它要么是 `true`,要么是 `false`。
- 程序通过判断一个布尔值并运行不同的代码来做出选择。
- 在这节课里,你学习 `if` / `else if` / `else`,以及如何构造条件。

---

## 关系运算符

- 这些运算符比较两个值,并给出一个 `boolean`:
  - `<` 小于,`>` 大于
  - `<=` 小于或等于,`>=` 大于或等于
  - `==` 等于,`!=` 不等于
- 注意:`=` 把一个值放进变量里,而 `==` 是**判断**两个值是否相等。

---

```java
public class Main {
    public static void main(String[] args) {
        int score = 75;
        System.out.println(score > 50);   // true
        System.out.println(score == 100); // false
        System.out.println(score != 75);  // false
    }
}
```

---

## if / else if / else

- `if` 只有在它的条件为 `true` 时才运行那个代码块。
- `else if` 在前面的条件都为 `false` 时,再检查另一个条件。
- `else` 在**没有任何**条件为真时运行。

---

```java
public class Main {
    public static void main(String[] args) {
        int n = -4;
        if (n > 0) {
            System.out.println("positive");
        } else if (n < 0) {
            System.out.println("negative");
        } else {
            System.out.println("zero");
        }
    }
}
```

## 逻辑运算符

- `&&` 表示**并且**(and):只有**两边都**为真时才为真。
- `||` 表示**或者**(or):**至少一边**为真时就为真。
- `!` 表示**非**(not):它把 `true` 变成 `false`,把 `false` 变成 `true`。

---

```java
public class Main {
    public static void main(String[] args) {
        int age = 16;
        boolean teen = age >= 13 && age <= 19;
        System.out.println(teen);        // true
        System.out.println(!teen);       // false
        System.out.println(age < 13 || age > 19); // false
    }
}
```

---

## 短路求值

- Java 从**左到右**读 `&&` 和 `||`,并在能确定结果时提前停止。
- 对于 `&&`,如果左边是 `false`,结果就已经是 `false`,所以右边会被**跳过**。
- 对于 `||`,如果左边是 `true`,结果就已经是 `true`,所以右边会被**跳过**。
- 这让你可以在使用某个东西**之前**先检查它是否安全。

---

```java
public class Main {
    public static void main(String[] args) {
        int x = 0;
        // Left side is false, so x / x is never run (no error).
        boolean ok = (x != 0) && (10 / x > 1);
        System.out.println(ok);   // false
    }
}
```

## `==` 与 `.equals()` 的字符串陷阱

- 对于 `int` 和 `boolean`,`==` 工作得很好。
- 对于 `String`,`==` 检查它们是否是**内存中同一个对象**,而不是相同的字母。
- 两个字母相同的字符串仍然可能是**不同的对象**,所以 `==` 可能是 `false`。
- **永远**用 `.equals(...)` 来比较字符串。

---

```java
public class Main {
    public static void main(String[] args) {
        String a = "cat";
        String b = new String("cat");
        System.out.println(a == b);       // false (different objects)
        System.out.println(a.equals(b));  // true  (same letters)
    }
}
```
