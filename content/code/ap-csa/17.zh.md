## 继承:在一个类的基础上扩展

- **继承**(inheritance)让一个类复用另一个类的字段和方法。
- 被扩展的类是**父类/超类**(superclass);新建的类是**子类**(subclass)。
- 子类**是一种**(is-a)它的父类:`Dog`(狗)**是一种** `Animal`(动物)。

---

## `extends` 和 `super`

- 写 `class Dog extends Animal`,就让 `Dog` 成为 `Animal` 的子类。
- `Dog` 会自动得到 `Animal` 的每个**公有**方法,比如 `getName()`。
- 子类的构造方法先用 `super(...)` 调用父类的构造方法。

---

```java
public class Main {
    public static void main(String[] args) {
        Dog d = new Dog("Rex");
        System.out.println(d.getName());   // Rex   (inherited from Animal)
        System.out.println(d.speak());     // Woof  (Dog's own version)
    }
}

class Animal {
    private String name;
    public Animal(String name) { this.name = name; }
    public String getName() { return this.name; }
    public String speak() { return "some sound"; }
}

class Dog extends Animal {
    public Dog(String name) {
        super(name);          // run Animal's constructor
    }
    public String speak() {   // override Animal's speak()
        return "Woof";
    }
}
```

---

## 重写一个方法

- 子类可以**重写**(override)一个继承来的方法:用**相同的签名**,换一个新的方法体。
- `Dog` 的 `speak()` 对 `Dog` 对象来说,替换掉了 `Animal` 的 `speak()`。
- 它**没有**重写的方法(比如 `getName()`)仍然原样继承下来。

---

## `super.method()` —— 扩展,而不是替换

- 有时你想要父类的成果,**再加上**一点点。
- 在重写里面,`super.describe()` 会运行**父类**版本的方法。
- 然后你在它的结果上添加,而不用从头重写。

---

```java
public class Main {
    public static void main(String[] args) {
        Bird b = new Bird("Tweety");
        System.out.println(b.describe());
        // I am an animal named Tweety, and I can fly
    }
}

class Animal {
    private String name;
    public Animal(String name) { this.name = name; }
    public String describe() { return "I am an animal named " + this.name; }
}

class Bird extends Animal {
    public Bird(String name) { super(name); }
    public String describe() {
        return super.describe() + ", and I can fly";   // reuse, then add
    }
}
```

---

## 是一种(is-a):子类能用在需要父类的地方

- 因为 `Dog` **是一种** `Animal`,你可以写 `Animal a = new Dog("Spot");`。
- 变量的类型是 `Animal`,但对象其实是一个 `Dog`。
- 这条 "is-a" 规则正是下一课**多态**(polymorphism)的基础。
