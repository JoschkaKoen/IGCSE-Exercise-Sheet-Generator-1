## 一个盒子里装很多值

- 一个**数组**(array)按顺序保存很多个**相同类型**的值。
- 在此之前,一个变量只保存一个值。数组保存一整个列表。
- 例如,一个班的考试分数可以放在一个 `int` 数组里。

---

## 用值来创建数组

- 如果你知道这些值,就把它们列在花括号 `{ }` 里。
- 类型是元素类型再加 `[]`,比如 `int[]`。
- Java 会数你给了多少个值,并创建对应大小的数组。

---

```java
public class Main {
    public static void main(String[] args) {
        int[] scores = {90, 80, 70};
        System.out.println(scores[0]);   // 90
        System.out.println(scores[2]);   // 70
    }
}
```

---

## 下标位置

- 每个值都有一个**下标**(index,它的位置)。下标**从 0 开始**。
- 所以第一个值是 `scores[0]`,第二个是 `scores[1]`,以此类推。
- 你也可以**修改**一个值:`scores[1] = 85;`。

---

```java
public class Main {
    public static void main(String[] args) {
        int[] scores = {90, 80, 70};
        scores[1] = 85;                  // change the middle value
        System.out.println(scores[1]);   // 85
    }
}
```

---

## 创建一个空数组

- 用 `new int[n]` 来创建一个有 `n` 个值的数组。
- 每个 `int` 位置都会自动从 **0** 开始。
- 然后你一个一个地填进去。

---

```java
public class Main {
    public static void main(String[] args) {
        int[] nums = new int[3];   // {0, 0, 0}
        nums[0] = 5;
        nums[1] = 6;
        System.out.println(nums[0] + " " + nums[1] + " " + nums[2]);   // 5 6 0
    }
}
```

---

## `.length`

- `array.length` 告诉你数组里有多少个值。
- 注意:**没有圆括号** —— 是 `scores.length`,不是 `scores.length()`。
- 最后一个有效下标永远是 `length - 1`。

---

```java
public class Main {
    public static void main(String[] args) {
        int[] scores = {90, 80, 70};
        System.out.println(scores.length);            // 3
        System.out.println(scores[scores.length - 1]); // 70 (last value)
    }
}
```

---

## 用 `for` 循环遍历

- 要访问每个值,就让下标从 `0` 循环到 `length - 1`。
- 用 `i < array.length` 作为停止条件。
- 下面我们把每个值加起来。

---

```java
public class Main {
    public static void main(String[] args) {
        int[] scores = {90, 80, 70};
        int total = 0;
        for (int i = 0; i < scores.length; i++) {
            total = total + scores[i];
        }
        System.out.println(total);   // 240
    }
}
```

---

## 增强 `for` 循环

- **增强 for**(也叫 “for-each”)直接访问每一个值。
- 可以这样读:“对 `scores` 里的每一个 `int v`”。
- 当你只需要**值**、不需要下标时,就用它。

---

```java
public class Main {
    public static void main(String[] args) {
        int[] scores = {90, 80, 70};
        int total = 0;
        for (int v : scores) {
            total = total + v;
        }
        System.out.println(total);   // 240
    }
}
```
