## 字符串与嵌套循环

- `String` 是一排字符,每个字符都有一个**位置**(索引 index)。
- 第一个索引是 `0`,最后一个是 `length() - 1`。
- 在这节课里,你会一个字符一个字符地遍历字符串,并把循环放进循环里。

---

## 遍历字符串

- `s.length()` 给出字符的数量。
- `s.charAt(i)` 给出索引 `i` 处的字符。
- 让 `i` 从 `0` 循环到 `length() - 1`,就能访问每一个字符。

---

```java
public class Main {
    public static void main(String[] args) {
        String s = "code";
        for (int i = 0; i < s.length(); i++) {
            System.out.println(s.charAt(i));
        }
    }
}
```

---

## 数字符的个数

- 你可以数一个字符出现了多少次。
- 用一个从 `0` 开始的计数器。
- 单个字符用**单引号**,例如 `'o'`。用 `==` 来比较(比较的是字符 char,不是字符串)。

---

```java
public class Main {
    public static void main(String[] args) {
        String s = "balloon";
        int count = 0;
        for (int i = 0; i < s.length(); i++) {
            if (s.charAt(i) == 'o') {
                count++;
            }
        }
        System.out.println(count);   // 2
    }
}
```

## 构建一个新字符串

- 从一个空字符串 `""` 开始,用 `+` 添加字符。
- 从**末尾**读到**开头**,就能把字符串倒过来。
- 每个 `+` 都会生成一个更长的字符串。

---

```java
public class Main {
    public static void main(String[] args) {
        String s = "code";
        String result = "";
        for (int i = s.length() - 1; i >= 0; i--) {
            result = result + s.charAt(i);
        }
        System.out.println(result);   // edoc
    }
}
```

## 增强 for(预览)

- 遍历字符串时你通常需要索引,所以普通的 `for` 最合适。
- 但 Java 还有一种**增强 for**(enhanced for)循环,它直接访问每个元素。
- 你会在第 8 课和数组一起大量使用它。这里用文字做一个小预览。

---

```java
public class Main {
    public static void main(String[] args) {
        String[] words = {"hi", "there", "!"};
        for (String word : words) {
            System.out.println(word);
        }
    }
}
```

## 嵌套循环

- **嵌套循环**(nested loop)是一个循环里面又有一个循环。
- 对于外层循环的**每一步**,内层循环都会**完整地**跑一遍。
- 在下面,外层循环挑选一行,内层循环在那一行里打印星号。

---

```java
public class Main {
    public static void main(String[] args) {
        for (int row = 1; row <= 3; row++) {
            String line = "";
            for (int col = 1; col <= row; col++) {
                line = line + "*";
            }
            System.out.println(line);
        }
    }
}
```
