## 一个可以增长的列表

- 数组(array)的大小是**固定的**。一旦创建,长度就不能改变。
- **`ArrayList`** 是一种列表,它在程序运行时可以**增长**和**缩小**。
- 你必须在文件顶部**导入**(import)它:`import java.util.ArrayList;`。

---

## 创建一个 ArrayList

- 把元素的类型写在 `<...>` 里面。这是 AP CSA 的写法。
- `ArrayList<String>` 存放文本。`ArrayList<Integer>` 存放整数。
- 新建的列表一开始是**空的**,大小为 `0`。

---

```java
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<String> names = new ArrayList<String>();
        System.out.println(names.size());   // 0
        names.add("Ann");
        names.add("Bo");
        System.out.println(names.size());   // 2
        System.out.println(names);          // [Ann, Bo]
    }
}
```

---

## 你需要的五个方法

- `list.size()` —— 列表里有多少个元素。
- `list.add(x)` —— 把 `x` 放到列表的**末尾**。
- `list.get(i)` —— 位置 `i` 上的元素(第一个位置是 **0**)。
- `list.set(i, x)` —— 用 `x` 替换位置 `i` 上的元素。
- `list.remove(i)` —— 取出位置 `i` 上的元素;后面的元素会向左移动。

---

```java
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<String> fruit = new ArrayList<String>();
        fruit.add("apple");
        fruit.add("pear");
        fruit.add("plum");
        System.out.println(fruit.get(0));   // apple
        fruit.set(1, "grape");              // pear -> grape
        fruit.remove(2);                    // take out "plum"
        System.out.println(fruit);          // [apple, grape]
    }
}
```

## 数字需要 `Integer`

- `ArrayList` 只能存放**对象**(object),不能存放简单类型 `int`。
- 所以我们写 `ArrayList<Integer>`。`Integer` 是 `int` 的对象形式。
- Java 会替你在 `int` 和 `Integer` 之间转换。这叫作**自动装箱**(autoboxing)。
- 你可以 `add(7)` 一个 `int`,而 `get(0)` 会返回一个可以当作 `int` 使用的值。

---

```java
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<Integer> nums = new ArrayList<Integer>();
        nums.add(10);          // int 10 is boxed into Integer
        nums.add(20);
        int first = nums.get(0);   // unboxed back to int
        System.out.println(first + nums.get(1));   // 30
    }
}
```

## 两种遍历方式

- **下标 for 循环**使用位置 `0` 到 `size() - 1`。当你需要下标时用它。
- **增强 for 循环**(`for (int x : list)`)依次读取每个元素。当你只需要值时用它。
- 两种方式都会按顺序访问每个元素一次。

---

```java
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<Integer> nums = new ArrayList<Integer>();
        nums.add(3);
        nums.add(5);
        nums.add(8);

        // index for-loop
        for (int i = 0; i < nums.size(); i++) {
            System.out.println("at " + i + ": " + nums.get(i));
        }

        // enhanced for-loop
        int total = 0;
        for (int x : nums) {
            total = total + x;
        }
        System.out.println("total " + total);   // total 16
    }
}
```
