## 读写文本文件

- **文本文件**(text file)把文字存到磁盘上,程序结束后它还在。
- 在 Java 里**读**文件,我们用一个连到 `File` 的 `Scanner`。
- **写**文件我们用 `PrintWriter`。在这一课里,每个程序先**写**一个文件,再把它**读**回来。
- **附加内容:** AP CSA 考试**不**考文件输入输出。学它是为了写真实的程序 —— 考试不会考。

---

## 你需要的 import

- 文件工具放在两个地方,所以我们在顶部把两个都导入。
- `import java.io.*;` —— 用于 `File`、`PrintWriter` 和 `IOException`。
- `import java.util.Scanner;` —— 用于读取器 `Scanner`。

---

```text
import java.io.*;
import java.util.Scanner;
```

---

## 文件可能出错,所以我们写 `throws IOException`

- 读或写文件可能会出问题(文件可能不存在)。
- Java 要求你处理这种情况。最简单的办法是给 `main` 加上 `throws IOException`。
- 这句话的意思是"如果发生文件错误,就让程序停下来"。对这些任务来说这样就够了。

---

## 用 `PrintWriter` 写文件

- `new PrintWriter("data.txt")` 会创建(或清空)一个叫 `data.txt` 的文件。
- 用 `writer.println(...)` 写一行,就像往屏幕上打印一样。
- 写完后一定要调用 `writer.close()`,这样文字才会被保存。

---

```java
import java.io.*;

public class WriteDemo {
    public static void main(String[] args) throws IOException {
        PrintWriter writer = new PrintWriter("data.txt");
        writer.println(10);
        writer.println(20);
        writer.close();           // save the file
        System.out.println("wrote data.txt");
    }
}
```

## 用 `Scanner` 读数字

- `new Scanner(new File("data.txt"))` 打开文件来读取。
- 只要还有下一个整数,`in.hasNextInt()` 就是 `true`。
- `in.nextInt()` 读取下一个整数。读完后调用 `in.close()`。

---

```java
import java.io.*;
import java.util.Scanner;

public class ReadNumbers {
    public static void main(String[] args) throws IOException {
        Scanner in = new Scanner(new File("data.txt"));
        int total = 0;
        while (in.hasNextInt()) {     // more numbers?
            total = total + in.nextInt();
        }
        in.close();
        System.out.println(total);
    }
}
```

---

## 整行读取

- 有时一行是文字,而不是数字。那就改成读**整行**。
- 只要还有下一行,`in.hasNextLine()` 就是 `true`。
- `in.nextLine()` 把下一整行作为一个 `String` 读进来。

---

```java
import java.io.*;
import java.util.Scanner;

public class ReadLines {
    public static void main(String[] args) throws IOException {
        Scanner in = new Scanner(new File("data.txt"));
        while (in.hasNextLine()) {
            String line = in.nextLine();
            System.out.println("read: " + line);
        }
        in.close();
    }
}
```

- `hasNext()`(后面没有 `Int` 或 `Line`)检查下一个**单词**,`next()` 读取一个单词。
- 套路始终是:先用 `hasNext...` 方法**检查**,再用对应的方法**读取**。

## 在一个程序里先写后读

- 在每个任务里,你的程序先**写**一个文件,然后把同一个文件**读**回来。
- 这样你可以一遍又一遍地运行它,得到**相同**的结果。

---

```java
import java.io.*;
import java.util.Scanner;

public class RoundTrip {
    public static void main(String[] args) throws IOException {
        PrintWriter writer = new PrintWriter("data.txt");
        writer.println(3);
        writer.println(4);
        writer.close();

        Scanner in = new Scanner(new File("data.txt"));
        int total = 0;
        while (in.hasNextInt()) {
            total = total + in.nextInt();
        }
        in.close();
        System.out.println(total);   // 7
    }
}
```
