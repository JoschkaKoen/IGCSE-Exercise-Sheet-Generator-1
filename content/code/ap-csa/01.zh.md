## AP 计算机科学 A 使用 Java

- AP CSA 考试用 **Java** 语言。
- Java 很**严格**:每个程序都写在一个**类**(class)里面,并从一个叫 **main** 的方法开始运行。
- 你已经具备程序员的思维 —— 这里我们学习 Java 的规则。

---

## Java 的基本骨架

- 一个最小的 Java 程序就像下面的例子。你的代码写在 **main 里面**。
- `System.out.println(...)` 打印一行文字(就像 Python 的 `print`)。
- 每一条语句都以**分号** `;` 结尾。

---

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Java runs from inside main.");
        int year = 12;
        System.out.println("Grade " + year);
    }
}
```

---

## 类型

- Java 要求每个变量都有一个**类型**(type):`int`(整数)、`double`(小数)、`boolean`(真/假)、`String`(文本)。
- 类型写在前面:`int score = 10;`。
- 用 `+` 把文字和数值拼接起来:`"Grade " + year` 得到 `Grade 12`。

---

## 现在轮到你

- 每个任务都已经填好了类的骨架 —— 把你的代码写在 **main 里面**,或者补全给出的方法。
- 按**运行**来编译并运行,然后按**检查答案**。
- **第一次运行会加载 Java 引擎**,可能需要 10–20 秒。之后就会很快。
