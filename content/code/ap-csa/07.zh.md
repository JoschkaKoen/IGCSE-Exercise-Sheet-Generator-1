## 保护你的数据

- **封装**(encapsulation)的意思是隐藏一个对象的数据,只能通过方法来修改它。
- 这样能让数据保持安全。外部代码不能往里面放进一个错误的值。
- 主要的工具是 `private` 和 `public` 这两个词。

---

## `private` 字段

- 把实例变量标记为 `private`,这样类外面的代码就**不能直接接触它们**。
- 同一个类里面的方法仍然可以使用它们。
- 这是 AP CSA 的常规风格:字段都是 `private` 的。

---

```java
public class BankAccount {
    private double balance;   // hidden from outside code

    public BankAccount(double start) {
        this.balance = start;
    }
}
```

---

## 访问器(getter)

- 字段是私有的,所以我们加一个 `public` 方法来**读取**它。
- 返回字段的方法叫作**访问器**(accessor)或 getter。
- 它通常以 `get` 开头,比如 `getBalance()`。

---

```java
public class BankAccount {
    private double balance;

    public BankAccount(double start) {
        this.balance = start;
    }

    public double getBalance() {
        return this.balance;
    }
}
```

---

## 修改器(setter)

- **修改**字段的方法叫作**修改器**(mutator)或 setter。
- 它可以先**检查**新的值,并拒绝一个错误的值。
- 下面的 `deposit` 会忽略负数,所以余额保持安全。

---

```java
public class BankAccount {
    private double balance;

    public BankAccount(double start) {
        this.balance = start;
    }

    public void deposit(double amount) {
        if (amount > 0) {
            this.balance = this.balance + amount;
        }
    }

    public double getBalance() {
        return this.balance;
    }
}
```

## `static` 与实例

- 一个**实例**(instance)字段属于某一个对象。每个对象都有它自己的一份。
- 一个 `static` 字段属于**类本身** —— 它被所有对象**共享**。
- 下面,每当**任何一个** `Robot` 被创建时,`count` 都会加 1。

---

```java
public class Robot {
    private static int count = 0;   // shared by all robots

    public Robot() {
        count = count + 1;
    }

    public static int getCount() {
        return count;
    }
}
```

## `toString()`

- `toString()` 返回一段描述这个对象的文字。
- 当你打印这个对象,或用 `+` 把它拼接起来时,Java 会自动调用它。
- 它不接收参数,返回一个 `String`。

---

```java
public class Point {
    private int x;
    private int y;

    public Point(int x, int y) {
        this.x = x;
        this.y = y;
    }

    public String toString() {
        return "(" + this.x + ", " + this.y + ")";
    }
}
```
