## 创建你自己的类型

- 到目前为止,你用过 Java 提供的类,比如 `String` 和 `Math`。
- 现在你要**写自己的类**(class)。类是对象的蓝图。
- 一个对象保存**数据**(它自己的值)和**方法**(它能做的事)。

---

## 实例变量

- 每个对象保存的数据存放在**实例变量**(instance variable,也叫字段)里。
- 你在类的里面、但在任何方法的外面声明它们。
- 每个对象都有它**自己的一份**这些变量。

---

```java
public class Dog {
    String name;   // instance variable
    int age;       // instance variable
}
```

---

## 构造方法

- **构造方法**(constructor)用来初始化一个新对象。当你写 `new` 时它会运行。
- 它的**名字和类名相同**,并且**没有返回类型**(连 `void` 都没有)。
- 用 `this` 这个词来表示“这个对象自己的变量”。

---

```java
public class Dog {
    String name;
    int age;

    // Constructor: runs when we make a new Dog
    public Dog(String name, int age) {
        this.name = name;   // this.name is the field; name is the parameter
        this.age = age;
    }
}
```

---

## 实例方法

- **实例方法**(instance method)是一个对象能做的动作。
- 它可以读取并修改这个对象自己的实例变量。
- 你用点号在一个对象上调用它:`myDog.speak()`。

---

```java
public class Dog {
    String name;
    int age;

    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Instance method: returns a sentence about this dog
    public String describe() {
        return this.name + " is " + this.age;
    }
}
```

---

## 创建并使用一个对象

- 用 `new ClassName(...)` 来创建一个对象。这会调用构造方法。
- 把它存进一个变量里,变量的类型就是类名。
- 然后用点号调用它的方法。

---

```java
public class Main {
    public static void main(String[] args) {
        Dog d = new Dog("Rex", 4);
        System.out.println(d.describe());   // Rex is 4
    }
}

class Dog {
    String name;
    int age;

    public Dog(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String describe() {
        return this.name + " is " + this.age;
    }
}
```

---

## 为什么要用 `this`?

- 一个参数和一个字段可以有**相同的名字**(比如 `name`)。
- 在构造方法里面,`name` 表示参数;`this.name` 表示字段。
- `this` 永远表示**调用该方法的那个对象**。
