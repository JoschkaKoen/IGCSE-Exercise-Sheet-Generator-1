## 一个数值的网格

- 普通的数组是一**行**数值。
- **二维数组**(2-D array)是一个**网格**(grid):它有**行**(row)和**列**(column),就像一张表格或电子表格。
- 我们用它来表示棋盘、图像、成绩表等等。

---

## 创建一个二维数组

- `int[][] g = new int[3][4];` 创建一个有 **3 行**和 **4 列**的网格。
- 每个格子(cell)一开始都是 `0`(对于 `int`)。
- 你也可以在 `{...}` 里面直接用值把它填好。

---

```java
public class Main {
    public static void main(String[] args) {
        int[][] g = new int[3][4];   // 3 rows, 4 columns, all 0

        int[][] table = {
            {1, 2, 3},
            {4, 5, 6}
        };   // 2 rows, 3 columns
        System.out.println(table[0][2]);   // 3
        System.out.println(table[1][0]);   // 4
    }
}
```

## 读取和写入一个格子

- 一个格子需要**两个**下标:`g[row][col]`。先行,后列。
- 两个下标都从 **0** 开始。
- `g[1][2] = 9;` 把 `9` 存进第 `1` 行、第 `2` 列。

---

```java
public class Main {
    public static void main(String[] args) {
        int[][] g = new int[2][2];
        g[0][0] = 5;
        g[0][1] = 7;
        g[1][0] = 9;
        g[1][1] = 11;
        System.out.println(g[1][0]);   // 9
        g[1][0] = g[1][0] + 1;
        System.out.println(g[1][0]);   // 10
    }
}
```

## 网格有多大?

- `g.length` —— **行**的数量。
- `g[0].length` —— 第 `0` 行里**列**的数量。
- 在 AP CSA 中,每一行的长度都相同,所以 `g[0].length` 就是整个网格的列数。

---

```java
public class Main {
    public static void main(String[] args) {
        int[][] g = {
            {1, 2, 3},
            {4, 5, 6}
        };
        System.out.println(g.length);      // 2 rows
        System.out.println(g[0].length);   // 3 columns
    }
}
```

## 行优先遍历

- 要访问**每一个**格子,使用一个循环**嵌套在**另一个循环里面。
- 外层循环选择**行** `r`。内层循环选择**列** `c`。
- 这样会先完整地走完第 `0` 行,再走第 `1` 行,以此类推。这种顺序叫作**行优先**(row-major)。

---

```java
public class Main {
    public static void main(String[] args) {
        int[][] g = {
            {1, 2, 3},
            {4, 5, 6}
        };
        int total = 0;
        for (int r = 0; r < g.length; r++) {
            for (int c = 0; c < g[0].length; c++) {
                total = total + g[r][c];
            }
        }
        System.out.println(total);   // 21
    }
}
```
