## 一格一格的网格

- **二维数组**是一个网格:由同样类型的行和列组成。
- `int g[2][3];` 创建 2 行 3 列 —— 一共 6 个 int。
- 你用两个下标访问一个格子:`g[row][col]`。

---

## 先行后列(行优先)

- C 把网格**一行一行地**存:先存完第 0 行,再存第 1 行。
- 要访问每个格子,用一个 `for` 循环遍历行,再在里面用一个 `for` 循环遍历列。
- `g[r][c]` 是第 `r` 行、第 `c` 列的格子。两者都从 `0` 开始。

---

## 传递二维数组

- 把网格传给函数时,你必须给出**列数**:`int g[][3]`。
- C 需要列数才能在内存里找到每一行;行数可以留空。
- 所以函数写成 `int g[][3], int rows` —— 列固定为 `3`,行数则传进来。

---

```c
#include <stdio.h>

int main(void) {
    int g[2][3] = {{1, 2, 3}, {4, 5, 6}};
    int total = 0;
    for (int r = 0; r < 2; r++) {
        for (int c = 0; c < 3; c++) {
            total += g[r][c];
        }
    }
    printf("%d\n", total);   // 21
    return 0;
}
```

---

## 一次处理一行

- 你也可以只处理一行:对固定的行 `r`,只循环列。
- 这就是对某一行求和、或找出某行里最大值的方法。
- 把两者结合 —— 外层循环行、内层循环列 —— 就能处理整个网格。

---

## 现在轮到你了

- 在函数任务里,网格有 **3 列**,所以参数是 `int g[][3]`。
- **不要**自己写 `main` —— 检查器会提供。
