## C 没有 try/except

- 有些语言在出错时会**抛出异常**。C **没有**异常。
- C 的函数通过它的**返回值** —— 一个状态码 —— 来报告问题。
- 调用者在每次调用后检查这个状态码,再决定怎么做。

---

## 返回状态码

- 一个常见的规则:成功返回 `0`,失败返回一个负数,比如 `-1`。
- 所以 `int do_thing(...)` 成功时返回 `0`,做不到时返回 `-1`。
- 真正的“答案”则用另一种方式交回(见下一步)。

---

## 通过传出指针报告答案

- 如果返回值是状态码,那真正的结果放哪里?通过**传出指针**。
- 调用者传入一个变量的地址;函数用 `*` 把答案写进去。
- 失败时,函数返回 `-1`,并且干脆不去碰那个变量。

---

## 用哨兵值表示“没有值”

- **哨兵值**是一个表示“这里什么都没有”的特殊值:下标用 `-1`,指针用 `NULL`。
- 函数用哨兵值,好让调用者用一个简单的检查就能区分成功和失败。
- 在你信任结果之前,总是先检查哨兵值。

---

```c
#include <stdio.h>

int safe_div(int a, int b, int *result) {
    if (b == 0) return -1;          // failure: cannot divide by zero
    *result = a / b;                // success: hand back the answer
    return 0;
}

int main(void) {
    int r;
    if (safe_div(10, 2, &r) == 0) {
        printf("%d\n", r);          // 5
    }
    return 0;
}
```

---

## 现在轮到你了

- 成功返回 `0`,失败返回 `-1`,把真正的答案通过传出指针写回去。
- 失败时不要碰传出指针。**不要**自己写 `main` —— 检查器会提供。
