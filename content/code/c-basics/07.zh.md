## 一排盒子

- **数组**是一排同样类型的变量,在内存里挨着排放。
- `int a[5];` 为五个 `int` 腾出空间。它们是一整块,用编号来访问。
- 你可以在创建时就填好:`int a[] = {10, 20, 30};` 创建一个有三个元素的数组。

---

## 从 0 开始的下标

- 每个元素都有一个**下标**(位置)。**第一个**下标是 `0`,不是 `1`。
- `a[0]` 是第一个元素,`a[1]` 是第二个,在有 `n` 个元素的数组里 `a[n - 1]` 是最后一个。
- 读或写数组之外的位置(比如在 5 个元素的数组里访问 `a[5]`)是严重的错误。

---

## 用循环遍历

- 要访问每个元素,用 `for` 循环让下标从 `0` 走到 `n - 1`。
- 在循环里面,用 `a[i]` 读或改位置 `i` 上的元素。
- 这就是对数组求和、计数或查找的方法。

---

## 数组不知道自己的长度

- 数组**不**带着自己的大小。函数没办法问它“你有多长?”。
- 所以你总是把长度和数组一起传过去:`sum(a, n)`。
- 在参数里用 `const int a[]`,表示你保证只**读**这个数组,不去改它。

---

```c
#include <stdio.h>

int main(void) {
    int a[] = {10, 20, 30, 40};
    int n = 4, total = 0;
    for (int i = 0; i < n; i++) {
        total += a[i];
    }
    printf("%d\n", total);   // 100
    return 0;
}
```

---

## 现在轮到你了

- 把数组和它的长度 `n` 一起传进去,让下标从 `0` 循环到 `n - 1`。
- 函数任务里,**不要**自己写 `main` —— 检查器会提供。
