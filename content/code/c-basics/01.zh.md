## C 是一门编译型语言

- C 是最古老、最重要的语言之一。Linux、数据库,甚至 Python 都是用 C 写成的。
- 与 Python 不同,C 是**编译型**的:计算机先把你的代码变成一个程序,然后再运行它。
- C 很**严格**:每条语句都以**分号** `;` 结尾,每个变量都需要一个**类型**。

---

## C 的基本骨架

- 每个 C 程序都从一个叫 **main** 的函数开始。你的代码写在 **main 里面**。
- `#include <stdio.h>` 让你可以使用 `printf` 来打印文字。
- `main` 末尾的 `return 0;` 表示“程序没有出错,正常结束”。

---

```c
#include <stdio.h>

int main(void) {
    printf("C runs from inside main.\n");
    int year = 11;
    printf("Grade %d\n", year);
    return 0;
}
```

---

## printf 与类型

- `printf` 打印文字。文字里的 `\n` 表示“另起一行”。
- 要打印一个值,使用**占位符**:`%d` 对应 `int`,`%f` 对应 `double`,`%c` 对应单个字符。
- `printf("Grade %d\n", year);` 会把 `year` 的值放到 `%d` 的位置。
- 每个变量都要先写类型:`int year = 11;` 或 `double pi = 3.14;`。
