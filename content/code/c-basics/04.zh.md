## 重复做事

- **循环**让同一段代码运行很多次,这样你就不用手动复制它。
- C 有三种循环:`while`、`for`、`do-while`。它们都在**条件**为真时重复。
- 每一轮都会检查条件。当条件变为假时,循环就停止。

---

## while 循环

- `while (判断) { ... }` 先检查判断,如果为真就运行代码块。
- 代码块里必须有东西最终让判断变为假,否则循环永远不会结束。
- 当你事先不知道要重复多少次时,用 `while`。

---

## for 循环

- `for` 循环把三个部分放在一行:`for (起始; 判断; 步进)`。
- `for (int i = 0; i < 5; i++)` 表示:让 `i` 从 `0` 开始,只要 `i < 5` 就继续,每次加 `1`。
- 当你要数已知的次数时,用 `for`。

---

```c
#include <stdio.h>

int main(void) {
    int total = 0;
    for (int i = 1; i <= 4; i++) {
        total = total + i;     // accumulate: 1, then 3, then 6, then 10
    }
    printf("%d\n", total);     // 10
    return 0;
}
```

---

## 累加与计数

- **累加器**是一个变量,在循环过程中一点点积累出结果,就像上面的 `total`。
- **计数器**在每次发生某事时加 `1`,用来计数。
- `break;` 提前结束循环;`continue;` 跳到下一轮。

---

## 现在轮到你了

- 在循环**之前**建立累加器或计数器,在循环**里面**更新它,在循环**之后**使用它。
- 函数任务里,**不要**自己写 `main` —— 检查器会提供。
