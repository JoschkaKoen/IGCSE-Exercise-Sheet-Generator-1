## 把相关的数据组合起来

- **结构体**把几个值组合成一个东西。一个点有 `x` 和 `y`;一个矩形有 `width` 和 `height`。
- 里面的每个值叫做**成员**(或字段)。它们可以是不同的类型。
- 结构体让你把一个整齐的东西传来传去,而不是一堆零散的变量。

---

## struct 与 typedef

- 你先定义一次形状:`struct { int x; int y; }` 有两个 `int` 成员。
- `typedef` 给这个形状起一个简短的名字。写了 `typedef struct { int x; int y; } Point;` 之后,你就能写 `Point p;`。
- 在这些任务里,`typedef` 已经在起始代码里替你写好了。

---

## 用 . 访问成员

- 对一个结构体**值**,用点号:`p.x` 是 `p` 的 `x` 成员。
- 你可以像普通变量一样读它(`int a = p.x;`)和写它(`p.y = 5;`)。
- 一个函数可以构造一个结构体、填好成员,再 `return` 整个结构体。

---

## 用 -> 通过指针访问

- 当你有一个指向结构体的**指针**时,用箭头:`p->x` 表示“`p` 所指向的那个东西的 `x`”。
- `p->x` 只是 `(*p).x` 的简写。箭头更好读。
- 传入一个 `Point *` 能让函数**修改**调用者的结构体(就像对单个值的按指针传参)。

---

```c
#include <stdio.h>

typedef struct {
    int x;
    int y;
} Point;

int main(void) {
    Point p = {3, 4};
    Point *ptr = &p;
    ptr->x = 10;                    // changes p.x through the pointer
    printf("%d %d\n", p.x, p.y);    // 10 4
    return 0;
}
```

---

## 现在轮到你了

- 对结构体值用 `.`,有指向结构体的指针时用 `->`。
- 每个起始代码里都给了 `typedef`。**不要**自己写 `main` —— 检查器会提供。
