## 每个值都有类型

- 在 C 里,每个变量都有一个**类型**,说明它存放的是哪种值。
- 最常用的三种:`int`(整数)、`double`(带小数点的数)、`char`(单个字符)。
- 创建变量时必须写出类型:`int n = 5;` 或 `double pi = 3.14;`。

---

## 做算术

- C 用 `+`、`-`、`*`、`/` 做运算,就像计算器一样。
- `*` 和 `/` 比 `+` 和 `-` **先**算。用小括号 `( )` 可以改变顺序。
- `%` 运算符给出除法的**余数**:`17 % 5` 是 `2`。

---

## 整数的除法不一样

- 两个 `int` 相除时,C 会把小数部分丢掉:`7 / 2` 是 `3`,不是 `3.5`。
- 想保留小数,至少有一边要是 `double`。`7 / 2.0` 是 `3.5`。
- 可以用 `(double)` 把 `int` **强制转换**成 `double`:`(double)sum / count`。

---

## 读取输入

- `scanf` 把输入读进变量。`scanf("%d", &n);` 读入一个 `int` 到 `n`。
- 注意 `n` 前面的 `&`:`scanf` 需要变量的**地址**,才能把值填进去。
- 在这个练习场里,**Check answer(检查答案)**会喂入示例输入;只按 **Run(运行)**不会送入任何输入。

---

```c
#include <stdio.h>

int main(void) {
    int a = 7, b = 2;
    printf("%d\n", a / b);        // 3   (integer division)
    printf("%.1f\n", a / 2.0);    // 3.5 (one side is a double)
    printf("%d\n", a % b);        // 1   (remainder)
    return 0;
}
```
