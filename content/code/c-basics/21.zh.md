## 数字就是一串位

- 在计算机内部,每个 `int` 都是一排**位** —— 每个位是 `0` 或 `1`。
- 我们通常用十进制读数字,但机器把它们存成**二进制**(基数 2)。
- 按位运算符让你查看和改动单个的位。

---

## 位值与 2 的幂

- 在二进制里,每个位置都是一个 **2 的幂**:1、2、4、8、16……
- 字节 `00010100` 是 `16 + 4 = 20`。最右边的位是个位(1 的位)。
- 一个 8 位的字节能存 `0` 到 `255`。

---

## 用 >> 和 & 1 读取一个位

- **移位** `n >> i` 把所有位向右移 `i` 位,这样第 `i` 位就落到个位上。
- **掩码** `& 1` 只保留最低那一位,得到 `0` 或 `1`。
- 所以 `(n >> i) & 1` 读取第 `i` 位的值。

---

## 按位运算符

- `&`(与)、`|`(或)、`^`(异或)、`~`(非)把数字**一位一位地**组合起来。
- `<<` 和 `>>` 把位向左或向右**移动**。向左移 1 位会让数字翻倍。
- 这些是做掩码、标志位和底层技巧的工具。

---

```c
#include <stdio.h>

int main(void) {
    int n = 20;
    for (int i = 7; i >= 0; i--) {   // from the highest bit down
        printf("%d", (n >> i) & 1);
    }
    printf("\n");                    // 00010100
    return 0;
}
```

---

## 现在轮到你了

- 用 `(n >> i) & 1` 读取第 `i` 位,用循环查看每一位。
- `count_bits` 和 `get_bit` 的参数是 `unsigned int`。这两个**不要**写 `main`。
