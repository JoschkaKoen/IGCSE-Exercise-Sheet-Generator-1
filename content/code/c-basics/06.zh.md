## 循环里套循环

- 你可以把一个循环放进另一个循环**里面**。外层循环每走一步,内层循环都会完整地跑一遍。
- 这就是处理行和列、或者比较每一对元素的方法。
- 如果外层循环跑 `R` 次,内层跑 `C` 次,那么内层的代码体一共跑 `R × C` 次。

---

## 行与列

- 把外层循环想成**行**,内层循环想成**列**。
- 对于一块 `*`,外层循环选一行;内层循环打印那一行里的星号。
- 内层循环跑完一行后,打印一个换行 `\n`,开始下一行。

---

## 先拼一行,再拼一块

- `printf("*")` 打印一个星号,但**不**换行。
- 先打印完一行的所有星号,再用一个 `printf("\n")` 结束这一行。
- 每一行都这样做,行就叠成了一块。

---

```c
#include <stdio.h>

int main(void) {
    for (int row = 0; row < 3; row++) {
        for (int col = 0; col < 3; col++) {
            printf("*");
        }
        printf("\n");      // end the row
    }
    return 0;
}
```

---

## 用嵌套循环来计数

- 嵌套循环也能数“对”。要数满足 `i < j` 的对 `(i, j)`,就让内层循环从 `i + 1` 开始。
- 在内层循环里放一个计数器,把满足条件的对加起来。
- 注意起始值和结束值 —— 这里很容易出现差一(off-by-one)错误。

---

## 现在轮到你了

- 用不带 `\n` 的内层 `printf` 拼出一行,再用 `printf("\n")` 结束这一行。
- 函数任务里,**不要**自己写 `main` —— 检查器会提供。
