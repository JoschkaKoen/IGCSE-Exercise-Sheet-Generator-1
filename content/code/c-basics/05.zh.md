## 为什么要用函数

- **函数**是一段有名字的工作。你只写一次,需要时就**调用**它。
- 函数让程序更短、更清晰,也让你用小的工作拼出大的工作。
- 你已经用过 `printf` —— 它是 `<stdio.h>` 里的一个函数。现在你来写自己的。

---

## 参数与返回值

- 你传进去的值叫**参数**。你交回来的值叫**返回值**。
- `int max2(int a, int b)` 接收两个 `int` 参数,返回一个 `int`。
- `return x;` 会立刻结束函数,并把 `x` 交回给调用者。

---

## void 函数,以及用小拼大

- 不返回任何东西的函数,返回类型写成 `void`。它做事,但不交回值。
- 一个函数可以调用另一个函数。`max3` 可以调用 `max2` 两次来完成它的工作。
- 每个只做一件事的小函数,更容易读,也更容易改。

---

## C 会复制你的参数

- 当你调用函数时,C 给它每个参数的一份**副本**。
- 所以在函数里面改一个参数,**不会**改到调用者的变量。
- 记住这一点 —— 在后面的一课里,**指针**会让函数能够碰到原来的变量。

---

```c
#include <stdio.h>

void try_double(int x) {
    x = x * 2;            // changes only the COPY
}

int main(void) {
    int n = 5;
    try_double(n);
    printf("%d\n", n);    // still 5 — the original did not change
    return 0;
}
```

---

## 现在轮到你了

- 用给出的确切名字和参数写每个函数,并 `return` 正确的值。
- 你可以在一个函数里调用你的另一个函数。
- **不要**自己写 `main` —— 检查器会提供。
