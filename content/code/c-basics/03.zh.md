## C 里的真与假

- 默认情况下,C 没有单独的**真/假**类型。它用数字:`0` 表示**假**,**任何其他数字**表示**真**。
- 像 `5 > 3` 这样的判断会得到 `1`(真)。`5 < 3` 会得到 `0`(假)。
- (如果你 `#include <stdbool.h>`,就能写 `bool`、`true`、`false` —— 但普通的 `int` 在哪里都能用。)

---

## 比较与组合

- 用 `==`(等于)、`!=`(不等于)、`<`、`>`、`<=`、`>=` 来比较值。
- 小心:`==` 判断是否相等;单个 `=` 是**赋值**。两者不一样!
- 用 `&&`(**与**)、`||`(**或**)、`!`(**非**)来组合判断。`a && b` 只有在两者都为真时才为真。

---

## if / else if / else

- `if (判断) { ... }` 只在判断为真时运行这个代码块。
- 加 `else if (...)` 检查另一个判断,加 `else` 表示“以上都不是”。
- 只有**第一个**满足的分支会运行。

---

```c
#include <stdio.h>

int main(void) {
    int score = 72;
    if (score >= 90) {
        printf("A\n");
    } else if (score >= 60) {
        printf("Pass\n");      // this one runs
    } else {
        printf("Fail\n");
    }
    return 0;
}
```

---

## switch:在多个值中选择

- `switch` 根据**一个**值(`int` 或 `char`)来选择分支。
- 每个 `case` 用 `break;` 结尾,这样它就不会掉进下一个 `case`。`default:` 接住其他所有情况。
- 当你要把同一个变量和许多固定的值比较时,`switch` 很清爽。
