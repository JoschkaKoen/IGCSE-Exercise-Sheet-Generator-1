## 文件让数据在程序结束后保留

- **文件**把数据存在磁盘上,所以程序停止之后它还在。
- 你通过一个 `FILE *` 来操作文件 —— 这是 `fopen` 给你的一个指针。
- 在这些任务里,你先**写**一个文件,再在同一个程序里把它**读**回来,所以一切都是自包含的。

---

## 打开与关闭

- `fopen("data.txt", "w")` 以**写入**方式打开文件(它会创建文件或清空文件)。
- `fopen("data.txt", "r")` 以**读取**方式打开。
- 用完后总是 `fclose(f)`,这样数据才会保存,文件才会被释放。

---

## 用 fprintf 写入

- `fprintf` 和 `printf` 很像,只是它的第一个参数是 `FILE *`。
- `fprintf(f, "%d\n", x);` 把数字 `x` 和一个换行写进文件 `f`。
- 每个值单独写一行,这样读回来时更容易。

---

## 用 fscanf 和 fgets 读取

- `fscanf(f, "%d", &x)` 读一个数字;成功时返回 `1`。
- `fgets(buf, size, f)` 读一**行**文本到 `buf`;到末尾时返回 `NULL`。
- 循环直到读取失败,就能处理整个文件。

---

```c
#include <stdio.h>

int main(void) {
    FILE *f = fopen("nums.txt", "w");
    fprintf(f, "3\n4\n");
    fclose(f);

    f = fopen("nums.txt", "r");
    int x, total = 0;
    while (fscanf(f, "%d", &x) == 1) {
        total += x;
    }
    fclose(f);

    printf("%d\n", total);   // 7
    return 0;
}
```
