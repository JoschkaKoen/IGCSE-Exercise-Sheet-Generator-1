# Exam PDF libraries

These directories hold Cambridge-style IGCSE PDFs used by **natural-language** mode (`resolve_natural_language` lists filenames for the model).

| Directory | Subject |
|-----------|---------|
| `physics/` | IGCSE Physics (0625) question papers, mark schemes, etc. |
| `computer_science/` | IGCSE Computer Science question papers and mark schemes |

Paths are configured in `extract_exercises/config.py` as `PHYSICS_EXAM_DIR` and `COMPUTER_SCIENCE_EXAM_DIR`.

**Licensing:** Past papers are © Cambridge University Press & Assessment. Use only in line with their terms; this repo stores them for personal/educational tooling convenience.
