# 9.2 Algorithms

1. Show understanding that an algorithm is a solution to a problem expressed as a sequence of defined steps
2. Use suitable identifier names for the representation of data used by a problem and represent these using an identifier table
3. Write pseudocode that contains input, process and output
4. Write pseudocode using the three basic constructs of sequence, selection and iteration (repetition)
5. Document a simple algorithm using a structured English description, a flowchart or pseudocode
6. Write pseudocode from: a structured English description, a flowchart
7. Draw a flowchart from: a structured English description, pseudocode
8. Describe and use the process of stepwise refinement to express an algorithm to a level of detail from which the task may be programmed
9. Use logic statements to define parts of an algorithm solution
