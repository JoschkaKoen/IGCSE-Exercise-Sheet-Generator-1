# 3.2 Logic Gates and Logic Circuits

1. Use the following logic gate symbols: **NOT**, **AND**, **OR**, **NAND**, **NOR**, **XOR**
2. Understand and define the functions of: NOT, AND, OR, NAND, NOR and XOR (EOR) gates
3. Construct the **truth table** for each of the logic gates above
4. Construct a **logic circuit** From: a **problem statement**, a **logic expression**, a truth table
5. Construct a truth table From: a problem statement, a logic circuit, a logic expression
6. Construct a logic expression From: a problem statement, a logic circuit, a truth table
