# 1.1 Quadratics

1. carry out the process of **completing the square** for a quadratic polynomial $ax^2 + bx + c$ and use a completed square form
2. find the **discriminant** of a quadratic polynomial $ax^2 + bx + c$ and use the discriminant
3. solve **quadratic equations**, and **quadratic inequalities**, in one unknown
4. solve by substitution a pair of **simultaneous equations** of which one is linear and one is quadratic
5. recognise and solve equations in $x$ which are quadratic in some function of $x$
